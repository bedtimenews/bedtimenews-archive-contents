# 

大家好，2026年1月13日星期二，欢迎收看1005期睡前消息。请静静介绍话题：

<font color = "indigo">2026年刚刚开始，影视圈老牌巨头华谊兄弟就贡献了新闻。老板王忠军手里的5.5%股份，被执行司法拍卖。现在王忠军、王忠磊兄弟手里所有股份，只剩下8.26%。

过去8年，华谊兄弟亏损82亿，资产负债率达到87.69%，欠债将近23亿。华谊兄弟曾经是中国赚钱前三名的影视公司，为什么亏成这样了？</font>

21世纪第三个十年，中国大企业突然出现巨额亏损，9成可能是投资地产项目失败。这个规律，对华谊兄弟也适用。

现在到百度搜索一下华谊兄弟和地产，就能发现全国几乎每个有点名气的城市，都有一座华谊兄弟小镇。比如说2017年，华谊兄弟在长沙大王山度假区拿了1000亩地，投资34亿，要打造意大利小镇。按照官方宣传，这个小镇一比一复刻水城威尼斯，建了35栋文艺复兴建筑。同一年，华谊兄弟又跑到济南长清区拿了800多亩地，表示要投资50亿，建设电影小镇。济南小镇2021年顶着疫情开业，没过一年就倒闭了。南京、郑州，都有类似项目，下场或者是倒闭以后找国资接盘，或者是亏损。

2018年，华谊兄弟还找了个强大的合作伙伴，一起开发电影小镇。请静静帮我读一段新闻：

> <font color = "indigo">在广东阳江市旅游发展大会上，华谊兄弟联手恒大签下电影城项目，预计投资总额达到375亿。自2011年转型以来，华谊兄弟落定地产项目已有至少20个，尤其小镇之风盛行的今年，华谊兄弟地产小镇不足一年就落下将近十个，圈了上万亩土地。
>
> 恒大与华谊兄弟计划，未来合作开发100个小镇，每个小镇项目，华谊都将获得1亿品牌费用加10%股份。</font>

在华谊兄弟看来，自己手里IP可以直接变现，所以到处圈地。到2018年，华谊兄弟储备了超过14000亩土地。

![图片](https://mmbiz.qpic.cn/mmbiz_jpg/Mo3q9dvIibMCHg8RqGfzTvUbKbM2U8pictibRZ5vQNkGYEMIpzia6d2ofCmCibPCynwtfUdePbgeMwhh7eFDuVQuAvg/640?wx_fmt=jpeg&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=1)

2013年，华谊兄弟投资了一部烂电影《私人订制》，里面有一个情节，扮演领导的范伟，问地产商，苏州阳澄湖重元寺边上的地要不要。就在同一年，华谊兄弟真的在阳澄湖岸边、重元寺正对面拿了400亩地造主题乐园，表示要做中国迪士尼。把真实项目拿到电影里调侃，华谊兄弟当时的自信已经溢出屏幕了。

我就住在苏州工业园区，距离阳澄湖不远，经常路过华谊兄弟这个项目，从来没看到停车场装满1/4的车。至于说原因，我读一下娱乐项目列表，观众可以自己判断，愿不愿意为了这些项目花钱。我给大家念一段：

> 90年代老纽约风格的星光大道，夜晚星光闪闪；非诚勿扰区夜色撩人，添加浪漫气息；集结号区民国街道，夜幕降临歌舞升平；白天平静的陈家沟，入夜变身朋克太极范儿，嗨翻夏夜；通天帝国神都洛阳，大唐盛世，一派繁华气象。

简单说，华谊兄弟建了一些风情街或者仿古建筑，打上《非诚勿扰》、《集结号》的名头，就想让游客买单。事实证明，华谊兄弟的IP，根本没什么吸引力，连回本都困难。2019年疫情还没来，上海迪士尼的营收是70亿，华谊兄弟的阳澄湖游乐园不到3500万营收，有200倍的差距。等到2022年，地产项目开始纯亏钱，只给华谊兄弟贡献了47万收入。

华谊兄弟的供应商最早发现问题，2021年底，苏州主题乐园的工程建设方就找到法院，要华谊兄弟支付工程款。华谊兄弟拿不出钱，直接导致名下1.5亿资产被查封。去年5月，阿里拍卖突然出现一条信息，表示华谊兄弟苏州华谊项目需要还债，对外招募意向投资人。到了11月，一家叫作海合安的投资公司突然宣布，自己作为华谊兄弟债主，全资收购苏州项目，主题乐园也改名为“海合安苏州阳澄半岛乐园”。华谊兄弟真把阳澄湖项目做亏了。

<font color = "indigo">华谊兄弟已经在影视圈做到了龙头，为什么还要冒险去做旅游地产？</font>

因为在中国拍电影，没有想象中赚钱。另外一方面，华谊兄弟在中国影视圈的地位，也没有很多人想象得那么稳。

先说电影产业链，把片子拍出来只是其中一环，真正决定收入的，很多时候不是电影质量，而是院线数量和排片情况。所以全世界电影市场，都是发行商和电影院拿大头，制片公司赚不到多少钱。“渠道为王”这句话，在电影行业尤其适用。

就拿2016年中国电影票房冠军《美人鱼》来说，总票房34亿，扣掉税以后的钱，院线要分成将近57%，留给制片公司13亿出头。扣除《美人鱼》制作成本将近4亿，9亿的毛利润，22家参与投资的公司来分，周星驰最多也就拿到1亿出头。2018年，北京文化投资《战狼2》，电影拿到56亿票房，最后分到手利润不到两亿。

刚才介绍的两部电影，都创造过票房记录，所以制片方能赚到一些钱。但是整个中国电影市场，亏损率将近70%。这么算下来，绝大多数中国制片公司，赚的都是辛苦钱。所以华谊兄弟想把名气变现，还是盯上了房地产行业。

<font color = "indigo">掌握渠道的发行商拿走大多数钱，是中国特色吗？</font>

这里介绍一个概念，叫作“大片场”模式，专门形容上世纪40年代好莱坞发行模式。简单来说，米高梅、派拉蒙这些电影巨头同时拥有电影制片厂、发行公司和电影院。全美国市中心地段将近20%电影院，都在大企业手里，可以拿到全美将近三分之二票房收入。

除了用自己的电影院赚钱，大公司还可以欺负其他小电影院赚到超额利润。比如说今天很多电影迷喜欢看的恐怖片，一开始属于B级片。大公司用很低成本快速拍摄出来，跟自己的大片捆绑在一起，要求小电影院打包购买。如果小电影院想要第一时间拿到大片播放权，还在拍摄期就要提前付款，就算电影质量不好最后亏了钱，也跟制作公司没有关系。

为了打击美国电影巨头的垄断地位，小罗斯福从上任开始，就对几家巨头提起了反垄断诉讼，只是因为二战爆发，暂时停止诉讼。等到二战结束，美国联邦法院正式裁定“大片场”制度属于垄断，要求电影巨头卖掉所有电影院，这就是“派拉蒙法案”。

但是很快，电影巨头又找到了新的垄断方式，跟电视台和录像带公司合并，组建传媒巨头。比如说华纳跟时代集团合并，组成时代华纳集团，派拉蒙跟美国最大的电视网CBS合并。等到2020年美国宣布《派拉蒙法案》作废，现在美国最赚钱的电影公司，仍然延续了“大片场”模式。像网飞和迪士尼频道，都是自己制片，只在自己的流媒体频道播放。

类似美国的“大片场”模式，在全世界都是主流。就拿日本来说，东宝株式会社在拍电影同时，还运营超过350家电影院，同时还入股了富士电视台。

香港情况也差不多，我们80后比较熟悉的厂牌，比如说邵氏、嘉禾，都在香港本土和东南亚拥有大量院线。邵氏直接模仿好莱坞传统手艺，在清水湾建设制片厂，流水线生产电影。嘉禾换了一种做法，支持李小龙、成龙这些明星成立小型工作室，等片子拍出来再一起分钱。

![图片](https://mmbiz.qpic.cn/mmbiz_jpg/Mo3q9dvIibMCHg8RqGfzTvUbKbM2U8pictJDHclcPewJ67ngxe5ku9elu4OXtqzvYI2tddiaqS5RNVlACUezuA3IQ/640?wx_fmt=jpeg&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=2)



到了80年代，电影明星开始要求高片酬，东南亚国家保护本国电影产业，限制港片进口。院线资源保不住，香港电影巨头就撑不下去了。这个时候邵逸夫果断换了赛道，重点经营TVB，原来的电影城改成电视城，用来拍电视剧。TVB用电影资源拍电视剧，很快就碾压了同时代竞争对手。嘉禾没能及时调整赛道，最后只能破产。



美国、日本和香港的例子都说明，电影公司能不能赚到钱，跟电影质量关系不大，重点是能不能控制院线。如果不能控制院线，转头去控制电视台，也算是替代方案。



中国商业电影起步比较晚，上世纪90年代之前，北影、上影这些电影厂根本不考虑销售问题。电影厂拍出片子来，按固定价格卖给中影，中影再把拷贝发到各个省市电影公司。票房怎么样，跟制片厂没有任何关系。等到1993年，国家发布了《深化电影行业机制改革若干意见》，改变了电影制片厂盈利方式。从此以后，国产片的制作方要自己找地方电影公司，谈价钱找市场。进口电影还是保持特许经营，由中影公司负责。



这段时间中国电影市场还不大，电影院基本都是国有企业管理。制片厂拍出来的电影能不能放，重点不是价格，更不是质量，而是关系。中国第一家民营电影发行企业博纳，老板于东曾经是北京电影制片厂员工，也是韩三平老下属。后来有媒体介绍过于东早年经历，请静静帮我读一段：

> <font color = "indigo">于冬最初是北影厂普通的发行员，主要工作就是到全国的各条院线、影院发行北影厂的电影。现在的影院和发行有比较固定的分账模式，当时就靠发行员卖拷贝，谈票房分成。“过去是要靠定拷贝卖钱，基本上每去一个地方就是喝酒，一杯酒就定一个拷贝，就是1万块钱。你多定我几个拷贝，我就多喝几杯酒，”于冬回忆说。
>
> 陈凯歌和于冬跑了几天就感慨：“于冬挺有意思的，每天背着个拷贝哪儿都跑，哪儿都一顿大酒，从早到晚，三顿酒，谁受得了？这叫推销员之死。”</font>

华谊兄弟当年起家，也靠院线关系。王中军两兄弟，都是北京干部家庭出身，哥哥曾经是国家物资总局出版社做摄影记者，弟弟是北京物价局干部。做电影之前，华谊兄弟首先是一个广告公司，通过给中国银行、国家电力这些大型国企做广告，从电视台手里拿到不少广告时段。

为了扩大广告市场，华谊兄弟又开始拍电视剧，比如说《罪证》、《心理诊所》。华谊兄弟当时卖电视剧播映权，不要电视台付钱，而是要拿广告时段来换。通过这种方式，华谊兄弟跟全国很多大城市的文化部门都建立了联系，拥有了院线渠道，开始联合冯小刚拍电影。

刚才提到，电影公司想赚大钱，只有两种渠道，或者是拥有自己的电影院，或者是办电视台。归根结底，就是要给自己的产品找到地方去放。但是中国还有另外一个本土特色，就是不允许私营电视台存在，电视台领导同时也是国家干部。所以，内容制作方跟电视台紧密合作，还容易被牵扯进腐败案件。

比如说中国另一家影视公司小马奔腾，曾经出品过《黄金大劫案》、《心花路放》、《太平轮》。小马奔腾起家经历，也跟华谊兄弟差不多，先拿到央视广告代理权，然后逐渐开始参与影视剧制作。等到2013年小马奔腾准备上市，老板李明直接被逮捕，协助调查当年央视腐败案。

华谊兄弟很早就决定，把前途押在院线上边，到了2010年左右，已经拥有四百多个影厅。当时中国一共只有5000多个电影厅，华谊可以占到10%。但是时代很快发生了变化。2010年以前，中国电影市场规模不大，所以华谊还算得上有实力，有渠道。但是跟其它行业巨头的实力比，华谊这点资本不算什么。

2006年开始，万达为了给自己的商业综合体增加吸引力，开始投资电影业，一年时间就建了120个电影厅。到了2015年，万达电影厅的数量翻了将近20倍，达到将近2600个。同一年，中国拥有超过3万个电影厅，对比五年前增长了6倍。当时一个IMAX电影厅的造价300万左右，普通电影厅造价100万。这一年，整个华谊的利润不到10亿，全部拿来建电影厅，也没法跟万达相比。

因为中国没有派拉蒙法案，所以资本快速复制了美国的“大片场”模式。比如说万达，从2015年开始，陆续投资了《夏洛特烦恼》、《唐人街探案》和寻龙诀。另外，虽然中国电视台不允许私营，但是流媒体平台可以。所以2014年阿里巴巴也成立了影视公司，投资了《碟中谍5》、《星际迷航3》。同一时间腾讯影业成立，跟万达一起合作拍摄了《魔兽》。

节目一开始提到，中国70%的电影都在亏钱。华谊兄弟的对手，有地产项目和互联网平台提供现金流，买得起院线，也办得起视频平台，更亏得起钱。但是华谊兄弟不一样，如果亏掉两三部电影，可能公司就要直接破产。比如说周星驰要拍《美人鱼2》，华谊兄弟投资了3个亿，结果片子现在都没上映。2018年，华谊兄弟又投资3亿拍《手机2》，结果崔永元举报范冰冰，所有投资全部打了水漂，整个公司完全靠借债运转。

如果华谊兄弟不搞地产项目，可能不会欠这么多钱，但也很难赚到钱。王家兄弟清楚这个道理，宁愿赌一把。请静静帮我读一段新闻：

> <font color = "indigo">华谊兄弟“赌徒式”投资，陷入巨大舆论风波，特别是高持股重资产投资。但王中磊坚信自己的选择，回应外界表示：“我们第一个做这个事情，初期很难取得所有人信任。我们就要自己努力，包括投入较大资金。”
>
> 为了证明自己的决策正确性，王中磊对苏州华谊兄弟电影世界强行“续命”，成立时拉来资方中国平安进场，随后引入民生信托，并质押了项目股权。
>
> 从2015年开始，王氏兄弟陆续质押华谊兄弟的股权进行融资，截止2018年7月，两人合计质押18.71亿股公司股份，价值21.97亿元。</font>

华谊兄弟当年为了找出路，还投资了两家游戏公司，现在看来反而是最划算的投资。2010年，华谊兄弟投资了1.5亿，成了掌趣科技第二大股东。2015年，华谊兄弟又花了19亿，做了英雄互娱第二大股东。

过去十几年，中国手游产品想赚钱，如果不算华南地区的买量小游戏，一般有三种方式。一种是背靠流量入口，复刻成熟游戏模式做产品，比如说吃鸡和《王者荣耀》。一种是借助老牌IP，赚品牌粉丝的情怀钱，比如说《三国》和《梦幻模拟战》系列。最后一种就是米哈游带动起来的精品二次元游戏，高投入，但是也有高回报。

华谊兄弟投资手游，刚好占了其中两种。比如说掌趣科技，手里有《拳皇98》和《一拳超人》IP，2020年营收将近18亿。同一时间中国手游市场总营收也就2700亿。英雄互娱虽然自己没有什么大牌产品，但是做过库洛科技大股东，也就是精品二次元游戏《战双》和《鸣潮》的开发商。过去两年，华谊兄弟全靠卖这些手游公司股份，把亏损控制在了22.4亿。如果当年王家兄弟没有投资手游，华谊现在可能已经破产了，不会到2026年才上新闻。

<font color = "indigo">督工你也提到，阿里和腾讯通过视频平台，建立了类似“大片场”的模式。网飞依靠流媒体平台，做到美国第一大媒体公司。华谊当年如果也搞一个视频网站，凭借自己的电影版权，会不会过得好一些？</font>

按照中国用户的付费习惯，可能华谊撑不到今天。

简单来说，网飞能在美国甚至更多国家成功，是因为发达国家都经历过有线电视的年代，用户已经习惯花钱看电视。美国家庭一个月的电视订阅费，基本都在80美元左右。如果想看HBO收费频道，还要额外再付15美金。网飞对比HBO，月费最低只要8美元，还提供更多内容，可以对有线电视进行降维打击。

办一个视频网站，除了自制节目要花钱，版权、带宽费用也都是成本，每年基本开销都要一百亿以上。就拿网络带宽来说，全国所有视频平台将近10%的成本，都是网费。2023年，B站网费就花了13.7亿。体量更大的抖音，每个月网费要花掉10亿。所以从2022年开始，包括B站在内的各大平台，都在减少带宽费用支出，代价就是降低视频码率。

腾讯视频和优酷背后，都有中国顶级互联网巨头输血，所以亏得起。华谊没有金主输血，如果全靠自己，最多也就做到爱奇艺的水平。爱奇艺现在情况怎么样，我给大家念一段数据：

> 2025年第三季度财报显示，爱奇艺总收入66.8亿元，同比下降8%，净亏损近2.5亿元，去年同期录得净利润近2.3亿元，由盈转亏。会员服务、在线广告服务、内容发行收入同比均出现下滑。

爱奇艺能够坚持到今天，背后也有百度的功劳。百度虽然衰落，但也不是华谊兄弟可以碰瓷的资本。更加值得参考的例子是PPTV。2013年，苏宁花了35亿人民币买下PPTV，随后三年亏了30亿，最后5亿卖了出去。

事实上，中国的所有文娱产业，几乎都依靠大企业输血或者政府间接承担亏损，才活到现在。腾讯和阿里亏钱做视频平台，本来就不指望平台赚钱，只是为了保证流量入口，给自己的核心产品引流。类似赔本业务还有不少，比如说阿里巴巴收购饿了么，到现在已经换了6个CEO，每年给阿里带来百亿亏损，市场份额一直没法突破30%。

但是每隔几年，阿里都要再砸点钱，搞一场外卖大战。今年的外卖大战，饿了么直接在淘宝APP单独开了页面，如果用户从淘宝下单，还能够领到更多红包。现在饿了么品牌直接改名淘宝闪购，说明阿里根本不在乎外卖业务做得怎么样。只要用户还记得淘宝品牌、愿意用支付宝付款，阿里巴巴就能在别的地方把钱赚回来。

不过，上周末最新消息，国务院反垄断反不正当竞争委员会办公室依据《中华人民共和国反垄断法》，对外卖平台服务行业市场竞争状况开展调查、评估。超级企业砸钱冲击其他行业的事情，以后可能会受到限制。如果反不正当竞争法落实到文化领域，未来中国的电影和其他视频市场可能会慢慢回归正常，逐渐培养用户的付费习惯，也给生产内容的企业更多的空间。但华谊兄弟这种急着越界赚钱的畸形企业肯定是等不到那一天了。
