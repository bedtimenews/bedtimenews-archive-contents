# 

大家好，2026年2月1日星期日，欢迎收看1013期睡前消息。请静静介绍话题。



<font color = "indigo">去年年底土耳其宣布，从2026年开始，对中国公民实行入境免签政策。

土耳其旅游资源丰富，是第二个对普通中国公民实行免签的北约成员国。现在去土耳其，会遇到风险吗？</font>



从去年到现在，可能是土耳其独立以来，外部环境最好的一年。无论是地缘政治或者镇压国内分裂势力，埃尔多安政府都取得了成果。同时，土耳其暂时还没有为这些成果背上太多的负担，经济问题有所缓和，眼下确实是一个去土耳其旅游的好机会。



先说邻国叙利亚。早在2019年，趁着叙利亚打内战，土耳其发动“和平之泉”行动，直接控制了叙利亚北部4000平方公里领土。随后土耳其继续扶植反对派武装，攻击叙利亚政府，想要推翻阿萨德政权。



当时的阿萨德政权是俄罗斯的中东代理人，还自以为得到了中国支持，土耳其拿不到多少地盘。但是俄罗斯在2022年自不量力，侵略乌克兰，陷入旷日持久的战争，甚至还丢掉了黑海制海权，丧失了向地中海大规模投放力量的能力。事实证明，一旦失去帝国主义支持，阿萨德政权连还手的能力都没有。2024年底，叙利亚反对派只用了12天时间，攻陷了叙利亚几乎所有重要城市，随后控制了首都大马士革，阿萨德自己逃亡俄罗斯。



反对派武装最大的一股势力，叫作“沙姆解放组织”，一直是土耳其重点扶持对象。土耳其除了提供资金和武器援助，还派出军事顾问帮忙指挥。2024年底，“沙姆解放组织”领袖朱拉尼还没宣布担任总统，土耳其已经派出外交部长，给朱拉尼提供国际承认。两个月后，朱拉尼宣布担任总统，马上出访土耳其拜访埃尔多安，宣布两国紧密合作。



朱拉尼虽然有个总统名头，但是政权并不稳定。在东部和南部，分别有库尔德人、德鲁兹人建立的割据政权，当年一起打天下的反对派武装，现在直接变成军阀。这些势力也有自己的外国靠山。朱拉尼的政权想要继续生存，必须长期配合土耳其的国际战略。



把叙利亚变成自己的附庸国以后，埃尔多安专心解决库尔德人问题。库尔德人是土耳其、叙利亚、伊拉克和伊朗四国交界区域的山区民族，本来没有特别强的民族意识。但是，随着周边形成民族国家，库尔德人自己也要求建国。尤其是土耳其库尔德人，完全是被土耳其的粗暴政策逼迫，产生了独立倾向。



从1984年开始，一个叫作库尔德工人党的组织发起武装叛乱，多次在土耳其境内制造恐怖袭击。土耳其想要控制叙利亚，其中一个原因就是为了打击库工党势力。



1999年，土耳其特工逮捕了库工党领导人奥贾拉，情况反而变差了。奥贾拉被捕前，库工党武装主要跟土耳其军队打山地游击战。等到奥贾拉被捕，库工党把目标转向土耳其大城市，频繁制造炸弹袭击。



到了2025年，土耳其跟库工党的斗争，算是告一段落。去年2月，被长期监禁的库工党领导奥贾兰发布声明，要求所有武装人员放下武器，自行解散库工党。3月份，库工党领导层宣布停火，表示会响应奥贾兰的要求。等到10月份，库工党从土耳其境内撤出了所有武装力量。



解决了境内的麻烦，下一个目标当然是叙利亚境内的库尔德武装。过去几个星期，叙利亚现在的政府军和土耳其人联手，又打了一场空地协同的闪击战，拔掉了叙利亚库尔德人的所有城市据点，以及少量油田。土耳其的库尔德问题，可能会从此告一段落。

刚才提到，土耳其能够控制叙利亚，是因为俄罗斯侵略乌克兰，在其他所有地区被迫战略收缩。所以在俄乌战争一开始，土耳其积极支持乌克兰，比欧盟和美国更早。在俄罗斯初期进攻基辅和哈尔科夫期间，乌克兰用来攻击俄罗斯后勤设施的TB2无人机，就是土耳其产品。



2022年9月，泽连斯基本人给TB2生产商颁发了勋章，感谢他们为乌克兰反侵略斗争做出的贡献。到现在，就算俄罗斯反无人机的技术提高了不少，TB2作战能力已经没那么重要，乌克兰还是土耳其无人机的主要买家。除了无人机，乌克兰早期使用的集束炸弹、装甲车，也都是土耳其提供。乌克兰安顿难民，土耳其出钱又出物资。



但是从去年开始，随着俄罗斯进攻能力不断衰退，土耳其对俄乌战争的态度，出现了变化。



虽然从历史上来说，土耳其跟俄罗斯一直是老对手。但是埃尔多安也清楚，俄罗斯不能太强，但也不能太弱。如果俄乌战争彻底摧毁俄罗斯的实力，土耳其就失去了一个重要筹码，对美国和欧盟来说失去了一大块“统战价值”。考虑到土耳其已经解决了叙利亚问题，继续削弱俄罗斯，显然不符合自己的利益。



另一方面，俄乌战争开始之前，埃尔多安跟普京私人关系不错。2016年，土耳其军队想要发动政变，第二天被埃尔多安政府平息。当时有传言说，就是普京政府及时传递了情报，让埃尔多安尽快撤离总统府，没有被叛军控制。



让埃尔多安转变态度，更加直接的因素，是俄乌战火烧到了土耳其。从2025年开始，乌克兰重点打击俄罗斯能源产业，其中一个目标是黑海油轮。去年11月，靠近土耳其海岸线的两艘油轮爆炸起火，土耳其外交部迅速发表声明，强调袭击事件发生在土耳其黑海专属经济区，影响了当地航行安全。12月12日，俄罗斯轰炸乌克兰在黑海的港口，又有一艘土耳其集装箱船被损坏。



只炸两艘船，还不至于让土耳其过于紧张。土耳其真正关心的，是自己的能源命脉，“土耳其溪天然气管道”。2024年，“土耳其溪”管道向欧洲出口天然气167亿立方米，其中86亿出口到匈牙利。土耳其的天然气高度依赖这条管道，同一年进口了210亿立方米，比欧洲还多了43亿。去年2月，乌克兰用无人机袭击俄罗斯的克拉斯诺达尔，正好也是土耳其溪管道起点，差点破坏了压缩泵。

土耳其是一个缺乏能源的国家，2/3的能源靠进口，所以土耳其比欧洲更害怕类似于“北溪”管道爆炸的事件，所以希望俄罗斯和乌克兰降低战争烈度。从5月份开始，俄乌代表在伊斯坦布尔举行了3次会谈，虽然没有明显成果，但也说明土耳其不满足于当世界列强的棋子，开始试探着上桌当棋手。



除了周边邻国，土耳其也没忘了在传统的伊斯兰国家扩张。自从卡扎菲垮台，利比亚事实上分裂成了两个政权。一个偏宗教化，占据了首都和西部港口城市，背后有土耳其撑腰，被称作的黎波里政权。一个偏世俗，背后有俄罗斯、埃及和阿联酋支持，控制了利比亚东部领土，也叫“国民军”。



从2020年开始，土耳其直接在的黎波里派遣驻军，帮助的黎波里政权对抗“国民军”。俄罗斯也出动军队支援“国民军”，跟的黎波里政权僵持。作为交换，土耳其拿到了东地中海油气勘探权，在利比亚建军事基地。历史上的奥斯曼帝国对利比亚领土就是松散管理，现在埃尔多安对利比亚的影响力，可能还要超出当年的奥斯曼苏丹。



2020年的睡前消息[第126期](/main/101-200/126)介绍过利比亚冲突情况，当时我说，俄罗斯和土耳其很可能会在对抗的同时保持默契，维持利比亚的分裂。过去几年，利比亚确实还在分裂，谁也拿谁没办法。去年开始，也许是认为俄罗斯没有能力再干预地中海，土耳其开始跟“国民军”政权接触，想要通过和平手段拿到更多的利益。

[第1003期](/main/1001-1100/1003)节目提到，土耳其还在干涉另一个非洲小国索马里。土耳其早就在索马里建立了海军基地和陆军训练中心。去年最后一天埃尔多安宣布，要投资60亿美元，在索马里建立一个航天中心，弥补土耳其领土纬度太高的遗憾。这也是奥斯曼帝国曾经的红海扩张方向。



![0.png](https://mmbiz.qpic.cn/mmbiz_png/Mo3q9dvIibMCUic6tLDRe9t3FpSfdpoiauQp7IC3uic6QSYusibTkV5vkpiaz1gOJZKgmRTOEicuhvia3Ig2od4lhL4KdQ/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=4)

奥斯曼帝国只在非洲沿海扩张，现在的土耳其已经深入内陆，跟瓦格纳进行正面竞争。过去几年，土耳其在非洲设立了37个军事办事处，向24个国家出售武器装备。肯尼亚、乍得和尼日尔的军队，都接受土耳其训练，向土耳其开发军事基地。

除了俄罗斯，伊朗也是土耳其的老对头。当初叙利亚内战，伊朗和土耳其打了一场代理人战争。这两年伊朗日子不好过，埃尔多安除了幸灾乐祸，还有更深的考虑。伊朗继续乱下去，吃亏的可能是土耳其。

新年以来，伊朗一直在乱。官方承认的死亡人数已经超过3000。其中抗议最热闹的区域，就是西北部库尔德人聚集区。当地已经出现了部落武装，想要制造分裂。

![图片](https://mmbiz.qpic.cn/mmbiz_jpg/Mo3q9dvIibMCUic6tLDRe9t3FpSfdpoiauQsGWJtUqvMiabkXZYqhviaSqYs9LjOGvT7jaO3cBSVMV6YOMZBqypxMQA/640?wx_fmt=jpeg&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=5)

哪怕是奥斯曼帝国最强盛的时候，也没有越过两河领域，控制伊朗高原。埃尔多安很清楚自己不可能去切分伊朗领土。所以，伊朗不能太强，但也最好不要乱到解体，给库尔德人提供大片领土。1月12日，土耳其发言人公开表示，伊朗问题应该由伊朗政府解决，不希望伊朗出现任何混乱。

除了扩张领土和军事基地，土耳其也在外交方面获取支持。过去几年，土耳其跟巴基斯坦关系非常亲密。巴基斯坦的武装直升机、装甲车和军舰，不少都是土耳其货。前两天刚刚传出消息，土耳其准备加入沙特、巴基斯坦的战略防御联盟。联盟条约类似于北约，任何一个国家遭受攻击，等于所有国家都受到攻击。联盟如果真能落实，土耳其等于间接得到了一点核武器保护，对大国谈判也更有底气。

1月份土耳其交通部长宣布，英国飞机发动机制造商罗尔斯·罗伊斯投资土耳其，要在伊斯坦布尔建设欧洲最大的客机发动机维修中心。项目总计投资超过23.2亿美元，预计每年可以维修200台发动机，给土耳其航空公司创造15亿美元年收入。同时，项目还可以创造1000个专业技术岗位，帮助土耳其产业升级。


作为回报，土耳其去年和英国签署了一份军售协议，用80亿英镑买20架“台风”战斗机，价格明显偏高，有评论文章说，英国是按护卫舰的价格卖飞机。

土耳其重视发动机技术，是因为自己要卖战斗机。去年8月，土耳其跟印尼签署协议，准备用100亿美元的价格，向印尼出售48架自主研发的“可汗”战斗机。“可汗”战斗机现在用的发动机，还需要从美国进口。土耳其正在研发的国产发动机，还处于早期测试阶段。如果英国技术可以帮助土耳其发动机取得突破，花掉的钱很快就可以赚回来。最新消息，土耳其向西班牙出口了30架高级教练飞机，每架一亿美元。土耳其的航空工业已经从欧洲拿到回头钱了。



中国企业同样在土耳其投资。2024年，欧盟对中国电动汽车加征临时关税。比亚迪作为出口主力，可能被加税17.4%。土耳其同时趁火打劫，表示对中国进口汽车加税40%。为了避免制裁，比亚迪很快跟土耳其开启谈判，表示愿意投资10亿美元在当地建工厂，每年生产电车15万辆。作为回报，土耳其修改了关税决定，不再对比亚迪额外征税。

比亚迪快速妥协，是因为想借助土耳其进入欧洲，也因为土耳其拥有本土电动车制造商Togg，提供了谈判筹码。去年一年，Togg公司在土耳其卖了将近4万辆电车，比特斯拉和比亚迪都卖得好。

<font color = "indigo">前几年说到土耳其，经常有人提到埃尔多安的汇率政策，用降息来对付通胀。反常的手段被称为“埃尔多安经济学”。现在土耳其国内经济怎么样？</font>



虽然军事和外交领域有很大进展，但埃尔多安经济学已经失败了。



埃尔多安经济学的说法来自2021年，当时埃尔多安公开宣布，降息才能降低企业生产成本，减少通胀速度。从2021年到2023年，土耳其存款利率从14%降到了8.5%。央行行长实在看不下去，觉得必须加息。埃尔多安表示“不换思想就换人”，2年时间换了3位行长。



![图片](https://mmbiz.qpic.cn/mmbiz_jpg/Mo3q9dvIibMCUic6tLDRe9t3FpSfdpoiauQibxSZOd5jhUbsObexUuaYxvsicWdia0iccDkwctwHicyFJ3GpLj3293iatWA/640?wx_fmt=jpeg&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=7)



一开始埃尔多安运气还不错，俄乌战争开打以后，土耳其两头倒卖物资，赚到了不少现金，确实缓解了通胀问题。2023年，土耳其经济终于撑不住了，埃尔多安向经济学认输，央行把利率从8.5%紧急提高到50%。到了去年，土耳其存款利率还是停留在高位，达到38%。



但是船大难回头，到今年为止，土耳其通胀问题还是非常严重。去年12月，土耳其CPI指数同比上涨30.89%，全年平均通胀率将近35%。过去两年，土耳其通胀率多次逼近80%。同样的钱今天可以买肉，明天可能就只有买块面包。现在土耳其人一旦拿到现金，马上就会想办兑美元、欧元和黄金。一些小商户甚至不收本国货币，只要外币或黄金。所以，尽管土耳其对中国免签，但在物价和汇率方面，中国游客还是要当心。

除了高通胀以外，埃尔多安面临的最大问题是腐败。长期以来土耳其经济增长，和中国的类似阶段一样，靠基建拉动。当地一些大型工程，比如说伊斯坦布尔运河、大型机场，都被几家大型承包商瓜分。这些承包商的老板，几乎都是埃尔多安的亲戚。



2023年土耳其发生大地震，超过5万人死亡，30多万栋建筑受损或者倒塌。事后发现，建筑商当年为了节约成本，直接用海沙搅拌混凝土，钢筋规格也严重不达标。海沙含有盐分，持续腐蚀劣质钢筋，房子在地震面前一摇就倒。虽然埃尔多安政府逮捕了一些建筑商，但是当年负责房屋验收和审批的高级官员，根本不受影响。大地震的很多灾民，现在还被迫住在集装箱。



物价高，房子不安全，土耳其普通人就不会因为海外的胜利欢呼。2024年土耳其举行地方选举，埃尔多安领导的正义发展党遭遇沉重打击。当地几乎所有大城市，包括首都安卡拉和第一大城市伊斯坦布尔，都被反对派拿下。

为了缓和民意，埃尔多安被迫表示要自我检讨。但更重要的事情，是先下手为强。去年3月土耳其警察逮捕了伊斯坦布尔市长伊马姆奥卢，罪名是间谍罪。这个伊马姆奥卢，同时也是下一届土耳其总统大选的有力竞争者。



![图片](https://mmbiz.qpic.cn/mmbiz_jpg/Mo3q9dvIibMCUic6tLDRe9t3FpSfdpoiauQnHmeibj20wrxsK0Gs7icJ67Whh8LNmmLqqDQicgIIClUwGqSA9rVicYrDA/640?wx_fmt=jpeg&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=10)



虽然伊马姆奥卢否认了全部指控，去年底还是被检察机关起诉。按照检察官的说法，伊马姆奥卢给土耳其造成了将近40亿美元损失，可能要判最高2352年有期徒刑。



去年3月和9月，几万名土耳其人走上街头，抗议埃尔多安政府，有将近2000人被捕。到现在，埃尔多安还在不断抓捕反对派官员，已经有17名市长被逮捕。接下来一年，土耳其国内可能还会发生更多冲突，出现更多抗议活动。



对外领土扩张，不断建设新的海外军事基地，对内靠强硬镇压维持统治，中国人看到这种国家，首先会想到20世纪30年代的日本。从国际形势来看，埃尔多安政权面临的环境，也确实和二战前的日本有点像。



当时英帝国的全球秩序解体，给中等强国创造了扩张空间，现在美国也在放松对全球秩序的直接管理，让土耳其产生了上桌当棋手的想法。当时的英国和法国都在放松海外殖民利益，现在俄罗斯被拖在乌克兰，只能扔掉在中东和非洲的全部势力范围。所以，埃尔多安抓住机会，暂时忽视国内矛盾，利用土耳其的交通枢纽地位，在几个大洲同时撒钱，想在一定程度上重建奥斯曼帝国的势力范围，面对中美欧三大势力也有说话的权利。



但是，从棋子变成棋手，是要交门票钱的。维持海外势力范围，长期可能会有丰富的利益，但短期内必须有天文数字的投入。哪怕是占据中国领土十几年的日本帝国，在东北的投资都没能转化成二战期间的军事力量，反而消耗了投资，影响了国内的物资供应。



今天的土耳其政权也一样，以中等国家的国力，撑起一个大国的面子，而且扩张范围横跨几千公里，暂时看不到各处布局形成闭环产业链的时间点。所以，除非能快速抢到油田这种立竿见影的回报，埃尔多安的海外经营恐怕也就到此为止了。能把过去几年抢到的东西消化掉，都算是他的成功。
