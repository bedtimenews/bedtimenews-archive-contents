# 

大家好，2026年2月8日星期日，欢迎收看1016期睡前消息。



经常有观众在各个平台问我，类似于睡前消息，还有哪些中国媒体也关注中国问题。我的推荐名单前两名几乎是固定的，一个是新华社的《半月谈》，另外一个是中国通讯社办的《中国新闻周刊》。顺便说一下，新华社和中国新闻社，也是中国大陆地区仅有的两家通讯社。



比如说2026年第一期半月谈，至少能找到五六篇有参考价值的文章。这里摘录几段内容，其中一篇的标题是[《器材沦为废铁，球场野草齐腰——部分基层健身设施闲置浪费》](http://www.banyuetan.org/jrt/detail/20260116/1000200033134991768526675016187276_1.html)：



> <font color = "indigo">*西部某县曾将一处篮球场作为典型进行宣传报道，然而半月谈记者经过多方打听、辗转多条小路，最终在村民指引下，驱车十余分钟才在山沟深处找到这片场地：空地上，孤零零立着一个篮球架，篮板上印着“农民体育健身工程——中国体育彩票捐赠”字样，周遭荒僻少人。*</font>

> *半月谈记者沿途随机向村民打听，他们有的表示知道该篮球场，但因为太过偏远从没去过，有的根本不了解情况，只知道高速公路大桥下方有个山沟。看着半月谈记者向其展示的宣传报道图片，一些村民很费解，“山沟沟里啥时候修了个篮球场？”*

> <font color = "indigo">*类似的“粗糙规划”并非个例。比如相关部门要求农体工程“一场两台”(即建设一个混凝土标准篮球场，配备一副标准篮球架和两张室外乒乓球台)，但有的地方将篮球架放在村委会院子或者农户家门口，连场地标识线都没有，运动场直接变身停车场。*</font>

> *体育工程是关乎群众健康的民生工程，绝非建完了事的“面子工程”。贵州南部某村庄是当地对外宣介的美丽乡村建设典型，每隔数百米便设有健身器材或运动场地。乍一看配置充裕，实际问题重重。半月谈记者来到公路边一处健身场地看到，9件器材坏了4件，其中“太空休闲椅”一侧铁链已经断裂，椅子歪斜。走进邻近的个别村寨，半月谈记者发现有的健身器材早已损坏，不仅无法正常使用，还存在安全隐患。*

> <font color = "indigo">*部分规模较大的县级体育馆也存在闲置现象。有媒体报道，贵州某体育文化中心体育场占地面积4.2万平方米，场馆按照中超比赛标准建设，东西看台能容纳观众1.2万人，设有地下停车位1100个。但半月谈记者来到该场馆实地探访发现，体育场中央的足球场野草长了有半人高，外围塑胶跑道则千沟万壑，破损严重。“这个足球场好像从来就没投入使用过。”*</font>

下一篇的标题是[《突击办会，“消化经费”——高校年底“会海”调查》](http://www.banyuetan.org/jrt/detail/20260109/1000200033134991767940553246903517_1.html)：

> <font color = "indigo">*半月谈记者在多个高校媒体联系群看到，2025年11月以来学术会议数量陡然上升。“农林院校土木工程专业学科发展研讨会”“农林院校信息类学院院长论坛”……12月27日 ，半月谈记者浏览某“双一流”高校网站时，一连串的学术会议新闻映入眼帘，11月29日4场，12月20日3场，且都是周六。*</font>

> *“年底几个月每个周末都有会。”2025年11月的一个周末，南京一位青年学者飞到深圳某高校参加了一场学术会议，而与她同行的老师，还要接连两个周末在这所高校的两个学院各参加一场学术研讨活动，“疲于开会”成为她及身边同事年底的常态，“大家周末不是在开会，就是在去开会的路上”。*

> <font color = "indigo">*半月谈记者随机浏览江苏某“双一流”高校网站，统计了2025年以来该高校每月学术会议、论坛报道的频次发现，11月28日之前有13场，但11月28日至12月26日已达9场。*</font>

> *拉学生凑人头的现象也很常见。该论坛的会议手册显示，某分论坛应有119名参会人员，但实际上座率仅1/3左右，且坐在后排的以学生为主。“年底学术会议多，不少是在周末早上八九点开始。我们起床迟，还没吃早饭，就靠茶歇垫垫肚子，中午再去‘蹭’自助餐。”一名“双一流”高校学生告诉半月谈记者，同学之间把在学术会议上蹭吃蹭喝的行为自嘲是“学术蝗虫”或“学术八戒”。*

> <font color = "indigo">*突击花钱”浪费科研资源。受访专家介绍，我国财政预算执行存在明显的周期性特征，年底是支出进度的关键时间点。高校作为预算执行单位，必须在年底前完成预算执行，否则面临资金收回或未来年度预算额度被削减的风险。*</font>

> *多位受访高校教师介绍，年底突击举办学术会议，可趁机邀请学术期刊编辑、主编并支付相应劳务费，或为回馈此前所发论文，或为后期发文铺路，而一些手握项目评审大权的学科大佬也是重点邀约对象。这类学术会议由此演变为合谋套取科研经费的平台。*

> <font color = "indigo">*“学术期刊编辑和学科大佬参会时，主办方不敢怠慢，不仅要给劳务费，还要专人接送，好吃好喝招待，这些都是不成文的规定。”一名高校教师介绍，一场学术会议劳务费少则几千元，多则上万元，有的嘉宾还会在参会地区顺道一游。*</font>

上面的内容片段，都是我在半月谈自己的网站上摘录的。但半月谈的内容不是100%上网。比如说，同样是2026年第一期，《县级融媒体中心发展之困》这篇文章，在网上就不好找，所以我们订阅了实体版。这里也转载几段：

> <font color = "indigo">*A县融媒体中心2018年12月挂牌成立，是公益一类事业单位，核定事业编制87名。目前，该中心内设21个部室，下辖广电网络技术保障中心和广播电视信息网络传输有限责任公司，干部员工490人，其中编辑和记者共70余人。“60多名在岗在编人员整体年龄偏大，多数在40岁以上。”A县融媒体中心负责人说，“财政全供人员较少，400多人要靠单位挣钱养活。”*</font>

> *当前，虽然各地县级融媒体中心都在尝试创新传播手段和话语方式，如在社交媒体平台、短视频平台开设账号，但部分县级融媒体中心仍旧面临“官气比较大，人气上不去”的困境。一些粉丝十几万乃至几十万的县级融媒体中心官方视频号，其发布的内容点赞量时常只有个位数。“有时我们精心做一个视频，没什么人看；别人发一个几秒钟的东西，点赞量却很高。”C县融媒体中心一位制作人员说，“自我感觉不错，但是‘叫好不叫座’。”*

> <font color = "indigo">*部分受访干部表示，县级融媒体中心传播力、影响力不足的重要原因是公信力不足、贴近性不够。一方面，媒体行业形势不佳，少数无良人员打着“媒体”旗号，到基层变相“敲诈勒索”，损害了行业声誉；另一方面，少数地方“处置舆情时有钱摆平，建设阵地时束手束脚”，当地媒体的桥梁、纽带作用无法充分发挥，甚至陷入“谁看拍谁，拍谁谁看”的怪圈。*</font>

> <font color = "indigo">*今年4月，东部省份多位投资者反映，受城投公司国资背景以及高利率吸引，他们购买了一些城投公司发售的定向融资产品，投资金额从几十万元到数百万元不等，有些人押上毕生积蓄、养老钱，甚至借款投资。*</font>

> <font color = "indigo">*半月谈记者看到，上述广告页上清晰标注着外省某地级市的经济总量、经济增速，“助力地方经济，共享发展红利”12个红色大字格外醒目。地方国有平台公司相互担保，更打消了认购人的疑虑。*</font>

> *一位地方金融办监管干部透露，对于城投定融产品的异地交易，当风险达到一定等级，才能引用地方金融条例予以叫停。“被叫停时，外省份城投定融产品发行方还抱怨‘存在地方保护主义’。”*

> <font color = "indigo">*半月谈记者调研了解到，北方某市的一家城投公司成立于2024年7月，同年9月开始发售相关定融产品。多位认购人称，他们第一时间认购该产品，但时隔一季度左右，利息兑付就发生异常。*</font>



还有一篇11月的文章[《地方避债，让企业起诉但别上访？》](http://www.banyuetan.org/jrt/detail/20251124/1000200033134991763969501116258424_1.html)：



> <font color = "indigo">*近期，半月谈记者跟随某项目施工方负责人李成（化名）到西部某省份一基层政府部门讨要拖欠的账款。李成表示，如果问题久拖不决，将逐级上访甚至进京上访。该部门负责人对此较为紧张，在对李成安抚后建议其向法院起诉，以加快问题解决进度。*</font>



> *半月谈记者在宁夏、江苏等地调研发现，地方政府部门建议企业“可起诉但别上访”并非个案。一些政府部门或平台公司拖欠的账款金额较大，早先许下的偿债承诺多年难以兑现，面对中小企业讨债，相关干部往往建议企业起诉地方政府，依法依规解决相关问题。*



> <font color = "indigo">*企业经营中难免与地方政府打交道，企业起诉政府意味着“决裂”，以后就很可能再难与政府合作。江苏一家房地产开发企业负责人说，地方政府拖欠中小企业账款问题并不少见，但因拖欠账款起诉地方政府的企业比较少，因为企业还要在地方经营，“受制于人”。此外，由于部分项目层层转包，小企业一般挂靠在大企业名下，想成为诉讼主体，需要被挂靠企业盖章。实际上，大企业往往不愿支持小企业起诉地方政府。*</font>



> *北京市中伦律师事务所合伙人刘相文等人说，一些地方政府既当裁判员又当运动员，个别政府部门甚至以调查其他问题为由，变相对民营企业施压，企业受制于地方保护主义，面临“执行难”。*



> <font color = "indigo">*与此同时，法院长期面临案多人少情况，执行法官不足，易致程序拖延、执行延期。另外，最高法在多个案例中明确，冻结、扣划政府账户必须采取“审慎和严格的标准”，需用政府财政资金以外的自有资金清偿债务。一旦地方政府清偿债务缺乏积极性，企业在实际执行时就难以得到配合。*</font>

> <font color = "indigo">*不愿到基层任职的法官助理不在少数。问卷结果表明，高级法院法官助理有近七成愿在省高院入额，愿到基层法院入额的仅2人;中级法院法官助理愿到基层法院入额的还不到一成;即便是基层法院法官助理，愿到基层法院入额的也仅有六成多。*</font>

说完半月谈，再介绍一些《中国新闻周刊》的内容。可能是因为中国新闻社相对没那么“官方”，也可能是因为《中国新闻周刊》到了21世纪才正式出版。至少在标题方面，《中国新闻周刊》更直接，更有冲击力，适合给中国其他媒体做榜样。

比如说上周的文章：[《不拔管，百万赔偿就没了》](https://www.inewsweek.cn/kj/2026-01-30/28818.shtml)，请静静读一段：

> <font color = "indigo">*记者在中国裁判文书网以“超过48小时”“视同工伤”为关键词检索，共获得2000余份裁判文书，其中展示的前600份多为近年案件。在医疗抢救技术不断进步的背景下，“48小时条款”在实践中也将部分职工家属推入两难境地：如果希望获得工伤保险待遇，就要在48小时内放弃家人的治疗；如果继续救治，往往意味着超过48小时后无法享受工伤保险待遇。*</font>

> *中国政法大学民商经济法学院教授、社会法研究所所长娄宇向《中国新闻周刊》解释，“48小时条款”的立法初衷，本是为简化工伤认定。20世纪八九十年代，工伤认定中通常要求证明疾病与工作之间存在因果关系，但实践中极难操作，因为影响死亡的因素过于复杂。“后来，立法上就选择了48小时这样一个明确的时间节点。”在他看来，这一转变本质上是以形式标准取代实质判断，以追求同案同判。但对于突发性的严重疾病，如果在48小时内仍抢救无效，生存可能性已极低。即便将这一时限延长至100小时，争议依然存在，总会有人被卡在临界点。*

> <font color = "indigo">*由此，“48小时条款”争议背后，指向另一个更为根本的问题：在医疗技术不断进步的背景下，法律究竟应以何种标准认定死亡？医学上，死亡认定主要包括两种路径：一是心肺死亡标准，即以心跳、呼吸停止作为判断依据；二是脑死亡标准。在器官捐献领域，不少案例是以脑死亡作为死亡标准。在国内，2019年，《中华医学会》杂志发布《中国成人脑死亡判定标准与操作规范（第二版）》，在医学层面建立了较为成熟的脑死亡判定技术规范，但脑死亡尚未立法。*</font>
