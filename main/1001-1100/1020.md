# 

大家好，2026年2月20日星期五，欢迎收看1020期睡前消息。请静静介绍话题。

<font color = "indigo">1月底，台湾省统计部门发布报告，表示台湾省经济增长率达到8.63%，是过去15年最高。同时台湾省人均GDP达到3.95万美元，超过日本和韩国。

作为曾经的亚洲“四小龙”之一，台湾属于发达经济体，但是经济增速不仅超过很多老牌发达国家，甚至也超过了越南这种低收入国家。

过去一年全球经济形势都不太好，绝大多数国家和地区都跟美国有贸易纠纷，台湾为什么能高速增长？</font>

台湾省2025年的经济指标确实很好看。上次台湾经济增长率超过8%，还是2010年。当时受到全球金融危机影响，2008年台湾没有任何经济增长，2009年还跌了2%，2010年的数据只能算触底反弹，补上前两年的缺口。这次不一样，2024年台湾增长率已经达到了4.6%，没有补缺口的余地，8.6%真的是一年中平地涨起来的。

![图片](https://mmbiz.qpic.cn/mmbiz_png/VZDFliaW3RYyo8W0tsRo6k4yFB6qOn8ibxCqqyfkl9ibwnKQUUdicb262ucFPibTJPr1f9J6mlfDaB9EeaBaPibKFYwg9iccYkF3ibAgsybmQ3Tkohw/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=1)

在分析具体影响之前，先要看看台湾的数据有没有水分。过去很多年，台湾本地人都在质疑官方数据，认为统计部门把一些GNI数据算进了GDP。简单来说，GNI是“国民总收入”，对比GDP，多了台湾企业和个人的境外收入。

虽然台湾省统计部门一直没有正面回应质疑，不过在我的同事看来，就算8.63%有一些水分，应该也不多。就拿日本来说，作为全世界境外收入最多的国家，这几年GNI对比同期GDP，也就多了2%左右。美国也有大量境外投资，但是过去两年为了支付债券利息，GNI甚至比GDP要低。另一方面，台湾的统计口径并没有突然变化，就算GDP有百分之几的水分，也不影响前年到去年的增长率。这个8.63%的数据，没什么大问题。

在年底之前，对台湾的增长数据，世界最大多数分析机构都看走了眼，认为和前年差不多，也就是4%到5%。这些机构事后分析，台湾是靠出口换来了超额增长。

2024年，台湾总出口也就4000亿美元左右。到了去年，出口数据直接跨了两个台阶，达到6407亿美元，增幅达到35%。如果扣掉进口数据，台湾一年的纯出口数据增加了765亿，全年总额达到1538亿美元。

台湾只有2340万人，1165万劳动力。平均每人出口6572美元，每个劳动力出口1.32万美元。

2025年，台湾省的名义GDP是28.68万亿新台币，相当于9091亿美元。如果直接按名义GDP计算，去年台湾光是净出口数据，就贡献了7.9%的增长，占到总体增长的9成以上。当然了，GDP的实际数据，统计起来要复杂得多。比如说去年一个面包10块，今年涨到了12块。反映到名义GDP，就是增长了20%。统计部门核算实际GDP，就必须考虑涨价因素，挤掉水分。

台湾省统计部门很快发布了调整后的数据，8.63%的增长率，其中有6.63%来自新增出口提供。台湾省本地的投资和消费，只贡献了不到2%。按照这个数据，台湾的新增出口，可以解释四分之三的GDP增长。

新增出口暴涨，有清晰的解释，就是全世界尤其是美国AI投资在拉动。过去一年，美国从互联网巨头到创业公司，都在砸钱搞AI。目前想要在AI方面领先，最简单粗暴的方法，就是堆算力、买显卡。微软、谷歌、亚马逊和Meta四家企业来说，2025年初就宣布要投入3200亿美元，建设AI基础设施。

按照过往经验，企业的重大投资计划，一般是吹给市场听的，需要慢慢落地，很有可能投到一半宣布取消。所以去年一开始，几乎所有分析机构预测台湾经济增长，只给到了4%。等到六七月份，分析机构预测增长率，也只是提高到5%左右。大家显然错误估计了美国企业对AI的决心，也低估了AI的发展速度。按照市场估计，美国上市企业宣布的AI投资，已经有将近三分之一落地。具体到硬件市场，就是英伟达全年赚了1300亿美金，增长了115%。

美国公司抢算力，放大了台湾省的产业优势。台湾在芯片半导体领域，还算不上全面领先。比如说上游的芯片设计领域，联发科比不过高通和苹果。生产半导体的专业设备，台积电也要依靠日本和欧洲的供应商。半导体其中一个大类是存储设备，基本被美国和韩国垄断。只有芯片代工领域，台湾具有绝对领先地位，台积电一家企业，营收占到全球同行的一半以上。AI时代，谁也绕不过台湾。

现在台湾省统计部门已经发布了出口商品名录，出口的商品超过四分之三，都是电子和通讯产品。

2018年的时候，台湾省对美国出口总值，是500亿美元出头，占台湾省贸易比重不到12%。2024年对美国出口1114亿美元，去年达到1983亿美元，一年内增幅78%，占贸易总量比例也达到30.9%。隔了20年，美国再次超过大陆，成为台湾省的第一大出口市场。

2025年，大陆跟美国围绕芯片博弈，连带也提高了台湾对大陆出口。观众应该都知道，去年小米推出了自己设计的手机SOC芯片。这款芯片采用3纳米制程，就是由台积电代工。2025年以前，台湾省对大陆出口总金额连续降低了三年，去年强势反弹，增长超过13%。同一时期，大陆进口总金额增长还不到1%。12月底台积电刚刚宣布，中国大陆客户可以正常使用先进制程，腾讯、阿里、字节跳动今年都在疯狂投AI，2026年台湾对大陆出口可能还要继续增长。


<font color = "indigo">台湾省增长数据没多少水分，但是各个网络平台上的的台湾人，似乎根本不在意这个数据？还有台湾网友直接表示，“没感受到台湾经济有多好，只是物价更高了”。这是为什么？</font>

简单回顾下[第999期](/main/901-1000/999)睡前消息的观点。

> 今年还有一个常用词——K型经济。一条曲线已经不能形容整体的产业发展，不同的产业和社会群体，进入了严重分化的两条曲线。
>
> 比如说前面提到，AI拉动美国经济，3季度的年化增长率是4.3%，但大企业普遍裁员，普通就业市场越来越差。
>
> 中国也有类似现象，一方面，我家乡承德的客服经济故事讲不下去了，1到11月所有中国规模以上工业企业的利润增速为0.1%，最后这几个月明显在往下走的。另一方面，高科技产业，比如说精细化工、智能制造，都不缺钱，利润多投资也多，能保持肉眼可见的产业升级。

两个AI大国，都出现了K型经济曲线。台湾省经济增长全靠中美的AI订单，也必然要继承中国大陆和美国的经济特征。中国大陆和美国毕竟还有一些其他产业，包括发达的农业和资源产业，台湾的经济增长全靠芯片代工，产业的两级分化更明显。

还是用数据说话。普通人判断经济形势，主要看三个方面，分别是收入水平、就业率，以及物价或者说通胀指数。



依靠大陆的产业体系，台湾的就业率表现还不错，没有出现类似美国的“无就业繁荣”。最近几年，台湾省失业率一直低于4%，用主流经济学来评价，台湾算是保持了“基本充分就业”。去年台湾失业率也比较低，平均只有3.35%，看起来是经济高速增长应该有的样子。



通胀率，台湾省的表现就比较一般了。从2021年开始，台湾的CPI指数累计涨了将近10%。当然了，受新冠疫情影响，通胀在全球都是普遍现象，台湾的数据不算夸张。台湾人感受不好，主要体现在吃饭变贵了。过去5年，台湾食品涨价将近18%，接近五分之一，所以台湾人完全有理由抱怨物价高。



更大的问题是收入。按照台湾省统计数据，有两个指标可以反应收入问题，其中一个叫作“家庭收支调查”。简单来说，有点类似大陆的人均可支配收入，统计范围包括了工资、投资和社会福利。



我们先看人均可支配收入。2024年，台湾的平均数是39万元新台币，中位数是32.7万新台币。对比2021年，台湾人均可支配收入是35.1万新台币，中位数是30万新台币。三年时间，台湾人平均收入涨了11%，中位数涨了9%。数据看起来不错，但是没有考虑通胀问题。同一周期，台湾CPI涨幅是7.8%，如果扣掉这个数据，台湾人的平均收入只增长了3%，中位数增长也就1%出头。



上面的收入和通胀数据，到2024年为止。但是开头说的台湾经济高速增长发生在2025年，所以说服力还不够。到我们这期节目截稿为止，台湾还没有发布2025年的家庭收支调查报告，所以我们要用另外一个指标，也就是每月工资水平来评价台湾的生活水平。



台湾2025年的工资数据，目前还缺少12月数据，这里只拿1到11月数据，跟2024年同期对比。核算下来，2025年11个月，每个台湾人的平均月薪是63470元新台币，对比2024年，这个数据是61000元左右，涨了4.1%。如果扣掉同期1.6%的CPI涨幅，实际增长是2.5%。

如果对比全球其它国家，这个数据还算不错，但是经不起细看。首先，台湾的就业市场，以服务业为主。台湾省现在的劳动力人口是850万人，其中服务业人口占到500多万，工业人口不到340万。这里出现第一个差别，工业岗位去年平均涨薪5.7%，服务业只有2.9%。也就是说台湾就业的大头，没有享受到多少增长红利。

在工业岗位，也不是人人都赚钱。如果按行业细分，整个台湾省去年只有两个行业涨幅比较高，分别是电子零组件制造和电子光学产品制造。很显然，这两个行业跟芯片代工直接相关，但是在整个台湾省，这只提供了90万个就业岗位。这里出现第二个差别，芯片产业拉高了整个制造业工资水平。



接下来是第三个差别，芯片产业几家大公司，又拉高了整个芯片产业的工资水平。前面提到，台湾跟芯片代工有关的工作岗位，将近90万个，其中台积电提供了7万5千个，不到十分之一。2024年，台积电员工平均工资超过350万新台币，是同时期社会工资的将近5倍。联发科的工资比台积电还要高，达到431万新台币。



半导体巨头开出的工资水平，是社会平均工资5倍以上，结果台湾半导体岗位的平均薪资，只是所有工业岗位的2倍。这么算下来，除了几家半导体巨头，其它的电子产品组装厂，员工的工资也不会比其它行业高多少。



如果不考虑股市收益，目前台湾几家半导体巨头总体雇工规模，只有十多万人。把直系亲属也算进来，也就几十万人，不到台湾全省人口的2%。少数半导体公司，赚到了几乎所有经济增量，台湾人抱怨经济不好很正常。台湾上一轮经济高速增长，发生在上世纪80年代。当时台湾股市涨幅超过12倍，平均工资涨了10倍，当地人形容市场热钱太多，有一句土话：“钱淹脚目”。至少到目前为止，台湾这轮增长还没有淹到普通人的脚面。



当然了，如果全球继续投资AI，带动台湾出口始终保持高位，可以指望台积电员工在当地消费，溢出到其它行业。但是更多人担心，台湾经济过于依赖芯片产业，台积电和联发科赚到太多外汇，反而推高新台币汇率。到时台湾的传统制造业比如说高精度机床、高科技布料，就可能失去价格竞争力。台湾本土高端人才也可能全部流入半导体行业，导致其它产业衰落。



这种担心也不是杞人忧天，早就在很多地方发生过。就拿荷兰来说，1959年荷兰发现了储量巨大的天然气田，开始大量出口天然气。等到70年代荷兰发现，自己虽然赚到了很多外汇，结果制造业却倒闭了一大片，出现严重通货膨胀。经济学界还发明了一个专有名词“荷兰病”，用来形容出口繁荣、经济萎缩现象。制造业的利润当然比石油利润更健康，但如果不能向全社会扩散，短期内效果是一样的。



普通台湾人对经济悲观，体现到行为，就是不买或者少买大宗消费品。按名义汇率，去年台湾省的人均GDP已经是大陆三倍以上，人均可支配收入也在两倍左右。但是2025年台湾只卖出去41.4万辆新车，对比2024年下降了9.5%。如果说台湾没有新能源补贴，台湾人更爱买摩托车，但是摩托车销量也不乐观。2023年，台湾人买了87万台摩托车，去年数据虽然比2024年有回升，也只有80万出头。

无论如何，K型增长也是增长，总比停滞要好。90年代末到21世纪初，中国大陆沿海城市出口快速增长的同时，内地和东北出现大规模的工人下岗、农民抗税运动。但紧接着，K型增长带动了中国有史以来最大规模的收入增长和城市化。现在的欧洲和日本倒是没有K型增长，所有的行业都平等地停滞，所以很庸俗的右翼口号也能得到普遍支持。2025年4月以来，在AI和新能源产业快速进步的同时，中国大陆只有12月一个月的制造业PMI高于50%，其他9个月工业都处于收缩区间，海峡两岸的中国人都在担心K型增长，中国人应该从中看到危机，更要看到希望。

[第850期](/main/801-900/850)节目，我提出了“负农业税”概念，也就是对农村进行二次土地改革。对土地按面积、产量和位置收税，对农民按人头发钱。发出去的钱要多于收上来的税，但税不能不交。最后的效果，是残余的农业人口把耕地和宅基地交给现代农场，换一份体面的退休金。整个国家也摆脱农业内卷，不用依赖高额补贴去维持农业经济。

2026年初，清华大学管理学院教授曹远征也提出了类似的方案，宅基地换社保，请静静帮我读一段：

> <font color = "indigo">从长短期结合的角度出发，当前有必要一次性、较大幅度地提高俗称“农民养老金”的城乡居民基础养老金的最低标准。2025年这一最低标准仅为每月143元人民币。
>
> 在人口老龄化日趋严峻的情况下，提高最低标准不仅有助于扩大消费，也是维护社会稳定、长治久安的重要举措。正是在这个意义上，资金来源可考虑先由国债垫付，后由国有资产划拨变现解决。但最终的解决办法，还是在推进城市化的进程中，实现农民工的市民化。
>
> 其中，探索推进宅基地使用权换社保的改革十分重要。我国最低收入群体主要集中在农村，他们虽拥有宅基地上的房产，却没有相应的财产性收入。这个问题既制约内需扩大，也影响经济增长，希望在“十五五”期间能够有实质性突破。</font>

当时我说，在熟人社会解体的背景下，进城的老人继续孤独生活，缺乏精神安慰。在唯物主义文化退潮之后，退休的老人还是恐惧死亡，找不到生活价值。就算把所有的讲师送进监狱，榨干家产，退休老人也无非是换一个地方销毁自己的退休金，比如说宗教、烟酒甚至毒品。相比之下，当前的保健品产业可能还是一个最不坏的解决方案。
