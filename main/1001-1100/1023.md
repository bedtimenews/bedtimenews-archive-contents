# 

大家好，2026年3月1日星期日，欢迎收看1023期睡前消息。请静静介绍话题。



<font color ="indigo">2月8日，日本自民党在新一届众议院选举当中获得了创纪录的成绩，赢得465个席位当中的316个席位，占比超过三分之二。再加上维新会，执政联盟议席数量达到了352。高市早苗重新选举的赌博大获全胜，毫无悬念地再次当选日本首相。



随着高市早苗当选，日本“修宪”话题，再一次成为了舆论焦点，2月19日，高市早苗在内阁发布会上表示，希望尽快推动修改宪法和《皇室典范》的进程。



![图片](https://mmbiz.qpic.cn/mmbiz_jpg/VZDFliaW3RYyGsV0Jib6jjL4Fd0yX28cdrp8uPp6Oyy6iaia1JZ6BlHCbUiawWZxzFtVCyqNuZxodN85y9MPVSHkJfC9XxNbTnS1iaXT0TI55ZLsY/640?wx_fmt=jpeg&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=1)



督工，有关日本修宪的话题，似乎已经持续了很多年，这一次能成吗？</font>



自从1947年日本颁布和平宪法开始，日本右翼势力一直要求修宪，口号是“正常国家化”。但在20世纪，日本经济领先整个亚洲，和大多数亚洲国家的贸易关系都不错，可以做到“闷声发大财”，所以大多数日本人并不会认真考虑修宪问题。



到了21世纪，日本泡沫经济破裂，国际地位逐渐下滑，日本民众和政客普遍产生失落感，对当前政治秩序的认同感下降，日本社会进一步右倾化，“修宪”的支持率水涨船高。最近20年当政的自民党首相，大多数都有过修宪的想法。



不过，由于旧日本帝国主动侵略过所有周边邻国，所以日本修宪问题不是个简单的国内政治问题，而是东北亚甚至整个太平洋西岸的敏感话题，所以日本政客投鼠忌器，修宪往往停留在口头上。



从2012年到2020年，安倍晋三连续8年执政，他在表面上扮演一个强势的右翼领袖，多次公开声明要修宪，但是8年时间里，日本修宪进程原地不动，只是每次选举当口号，拿出来动员一下选票。



安倍晋三的做法不难理解，他的外公是前首相岸信介，父亲是前外交大臣安倍晋太郎，作为世家政客，作为当前政治秩序的既得利益群体，他从政的核心策略是维持日本的现有体制架构和利益分配格局，贸然执行激进方案并不符合他的需求。



2020年安倍下台，新冠疫情爆发、俄乌战争影响国际资源价格，日本经济长期停滞的问题开始暴露，安倍刺激经济的方案导致国内物价上涨，日本普通人开始右倾化，质疑和平宪法的合理性。民粹政党借着排外思想，开始拿到更多的选票，接过了安倍晋三的修宪口号。



在民粹选票的支撑下，比安倍晋三更加激进的右翼政客不断出现，而且往往是根基比较浅的新人，对现有体制缺乏敬畏感。高市早苗家里没有人从政，后来虽然和议员山本拓结婚，但两人复婚之后，山本拓改名高市拓，表示无条件支持高市早苗的立场。



根据日本媒体的调查，这一届81%的当选众议员赞同修改宪法，宣布自卫队是合法军队，和其他国家的军队有类似职能。就算高市早苗不当选首相，其他人在这个位置上，为了满足议员的诉求，也会提出差不多的政策。



![图片](https://mmbiz.qpic.cn/sz_mmbiz_jpg/VZDFliaW3RYw6OsH98Ul9sL1ZY8eDLOYC4f6UgdJTDZfYXHUdVp5ica2lib7VmDXa4tG7xsyArWGh8TVsgHaqIQSibexjYbZgF7Q1tyn47r3av0/640?wx_fmt=jpeg&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=2)



比如说，曾经和高市早苗竞争过首相位置的现防卫大臣、小泉进次郎，在记者会上公开表示，要完善以全民公投方式来修宪的环境。在自民党内，除了极少数的非主流议员之外，修宪议题已经形成共识。



![图片](https://mmbiz.qpic.cn/sz_mmbiz_jpg/VZDFliaW3RYzq9tD8MiaPUplhdfGoibhdsCibib9O0BYjnCANCNne8AbL5g8YIX2RWTAyTVkGyXiaUh5tOmkeUqzjFTsZDzjgUWYLL7careZur4ho/640?wx_fmt=jpeg&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=3)



高市早苗的修宪计划，主要有四个项目。



首先是新增紧急事态条款，允许政府在重大灾害、危机时发布紧急宣言，内阁可制定临时政令、延长国会任期。



其次，改革参议院选举制度；



第三、推动教育无偿化。宪法将明确规定从学前到高等教育都免费，推动机会平等。



然后，就是最重要的一条，修改宪法第九条。请静静帮我读一下日本当前宪法：



> <font color ="indigo">第九条 【放弃战争，战争力量及交战权的否认】
> 
> ① 日本国民衷心谋求基于正义与秩序的国际和平，永远放弃以国权发动的战争、武力威胁或武力行使作为解决国际争端的手段。
> 
> ② 为达到前项目的，不保持陆海空军及其他战争力量，不承认国家的交战权。</font>



宪法说 “不保持陆海空军及其他战争力量”，这导致日本自卫队在宪法体系下是“黑户”。高市早苗表示，可以保留第九条现有内容，但一定要加上一句话，规定自卫队的合法地位。



反对者认为，一旦自卫队取得法律地位，前面的“不保持战争力量、不承认交战权”内容就会弱化，而且还可能逐步增加更多的军事条款，最终消灭和平宪法。而支持者认为，给自卫队上户口，名实相副，如果连这一点都做不到，宪法未免自欺欺人，当然也谈不上“正常国家”。两方面都有自己的立场，而且往往是一方的激烈立场，让另一方面的立场更坚定。



<font color ="indigo">高市早苗的选举胜利，超出了所有人的预期，甚至大概率也超过了她自己的预期。现在她的首相位置已经稳固了，要修宪还有什么障碍呢？</font>



日本的议会是两院制，根据日本现行的法律，想要修改宪法，需要国会众议院和参议院各自以三分之二多数通过决议。通过之后再交给全民公投，得到50%以上的投票认可，最终完成。



这一次众议院选举大胜，同时自民党内进一步的右倾化，目前看来，修宪建议在众议院得到三分之二赞成票并不困难。但下一步参议院比较麻烦。



日本众议员任期四年，到期全部重选。就算不到期，内阁也可以解散重选，让议会体现最新的国民立场。这次高市早苗就行使了解散议会的权利，而且赌赢了。



参议院任期六年，每三年只改选一半，不到日期不能重选，体现的是之前几年的政治风向，首相也不能干预。



从制度设计来说，众议院代表当前的国民立场，所以权力大，预算案必须在众议院提出。首相虽然没有规定身份，但惯例是由众议员担任。



参议院的目标是维持政策的可持续性，提供稳定立场，所以权力较小。虽然可以否决法案，但众议院的大多数议案只要获得三分之二议员以上赞成，参议院只能否决一次，第二次再次有2/3的赞同票，绕过参议院直接生效。可以说，参议院发挥的主要作用，是让议员和全体国民“冷静一下”。



修改宪法属于长期发挥影响力的重大议题，所以“冷静决策”更重要，众议院不能强行通过，必须参议院也要有2/3的赞同，才能交给全民公决。



目前日本参议院一共有248个议席，2025年日本举行了参议院选举，在改选的124个席位当中，自民党只得到了39席，加上未改选的62席，一共101个席位。同时，最大的反对党立宪民主党，拥有38个席位。



![图片](https://mmbiz.qpic.cn/sz_mmbiz_jpg/VZDFliaW3RYywIduwnicqibBNYsibTiaHg2Owc7l7bT5g0rub8VodUcHfTXutzIIrIZBQPbGbLmP0a3icek0cDr9FicSZwicWWRGw1Lch2f6lXoo0IQ/640?wx_fmt=jpeg&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=4)



高市早苗上台之后，算上愿意拥护修宪议题的维新会、国民民主党、参政党和保守党等等政党，还是不能在参议院拿到三分之二多数所需要的166席。这就挡住了高市早苗的修宪想法。



参议院的地位终究低于众议院，而且也会定期改选，所以“制约”非常薄弱。现在“修宪派”已经很接近三分之二多数了。如果高市早苗还能够延续她目前的高支持率，最快在两年之后，下一次参议院选举，再多赢十几个席位，就可以扫清修宪障碍。当然，如果高市早苗的执政表现太差，那么形势也有可能被逆转。这就是参议院的价值所在。



前面提到，修宪的主要内容，是给自卫队合法身份。这说明，现实中的自卫队，和军队区别不大，而且地位不断上升。20世纪，自卫队的负责单位是防卫厅，在右翼政党推动下，2007年防卫厅升格成防卫省，负责人进入内阁。所以支持修宪的议员才提出了“名实相副”的诉求。



<font color ="indigo">日本修宪之后就会大规模扩军吗？</font>



很难，因为时间已经到了2026年，日本自卫队的最大限制不是法律，而是日本财政和人口压力。

 

回头看安倍晋三时代。虽然被称为右翼精神领袖，但是安倍晋三在军备问题上非常谨慎。



2019年安倍内阁的国家预算，军费5.2万亿日元，占预算总支出的5.2%，还不到日本GDP的1%，和其他国家比并不突出。同一年中国GDP正好100万亿，军费是1.19万亿。

安倍下台之后，日本军费开始连年上涨。2023年，岸田文雄内阁的国家预算，军费6.78万亿日元，占GDP的比例达到1.2%。

高市早苗上台之后，日本的军费增速再次提升。最新的2026年预算，军费接近9万亿日元，相比安倍时代几乎翻倍，达到了GDP 的1.5%。作为对比，中国2025年的比例是1.27%。

但是，高市早苗的军费，恐怕也很难再增加了。因为日本一直没有走出90年代之后的经济停滞。



安倍晋三执政期间，推动超宽松货币政策，逐渐摆脱了通缩，经济开始有起色。但是，经济增长，代价是用央行无限印钱，普通人承受物价上涨。扣除通胀因素，只看购买力，日本2025年的GDP只能说刚刚回到90年代高峰期水平。



![图片](https://mmbiz.qpic.cn/sz_mmbiz_jpg/VZDFliaW3RYz5iagovH1NWYKmyJ3FFyQweRicn8HoZXjXsxlzHNzvnAaGJE2glbnyxvsPxA3GHvLibl4gcPDLuQibbVQ1G770DXFJ0ckc6dVthHY/640?wx_fmt=jpeg&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=8)



与其同时，日本的财政状况可是比20世纪差多了。目前日本政府债务1145万亿，超过GDP的两倍。高市早苗内阁的最新预算，25%的支出用来还债，其中10.7%用来还国债的利息。今后几年，日本走出低利率时代，国债借新还旧的压力还会更大。

和90年代相比，日本的另一个问题是老龄化。高市内阁的预算，31%的支出用于社会保障，比军费的4倍还多。随着人口下降，高市早苗加军费的空间已经不多了。



目前高市早苗提出的扩军方案，最大动作，不过是把冲绳方面的一个旅，提升到一个师，实际上增加兵力一个团。和其他国家比，日本自卫队的师和旅编制都偏小，陆上兵力总共15万，最大的满编师也只有9000人。把一个旅改成师，实际增加兵力只有一两千。



此外，日本在作战飞机、主战舰艇方面也没有跨越式的升级方案，未来的计划只是配合美国的主战装备，在靠近台湾的岛屿上设置一些导弹阵地。如果日本在经济和人口两方面不能走出长期衰退问题，高市早苗的修宪计划，对日本的军队硬实力，不会有明显的影响。

接下来分析最新局势。

这个周末，美国和以色列主动进攻，在一年内第二次发动了对伊朗的战争。2025年6月那次，美国没有预先集结兵力，双方对攻还是打了12天。现在战争刚进入第二天，我就沿着逆时针方向，聊聊周边国家的变化。

伊朗东面有三个国家，阿富汗、巴基斯坦，隔海相望是大国印度。因为印度本土缺资源，长期以来印度一直是伊朗石油的大客户。在这次轰炸开始之前3天，2月25日，印度总理莫迪到了以色列，不仅去了行政首都特拉维夫，还公开访问了耶路撒冷，悼念哈马斯袭击的死难者。与其同时，印度和以色列展开自由贸易谈判。一般认为，以色列这次给印度提供了从来不出口的军事技术，比如说参加过实战的铁穹防空系统，换来了莫迪的坚定支持。

巴基斯坦和阿富汗，这几天大打出手。边境上普什图族地区双方陆军开火就不用说了，都宣布打死对方几百人。在美国攻击伊朗前一天，有视频为证，巴基斯坦飞机轰炸了阿富汗首都喀布尔。同时阿富汗方面也宣布空袭巴基斯坦首都伊斯兰堡，确实发生了爆炸，用的是飞机、无人机还是自杀炸弹就不知道了。

此外，去年3月份睡前消息[880期](/main/801-900/880)节目分析了巴基斯坦西南部的俾路支问题，结论是巴基斯坦的执政能力和经济实力都不足，化解不了内部族群矛盾，冲突会长期激化。但是，就在伊朗挨打之前几个星期，冲突有加强的趋势。

2月3日，俾路支武装同时攻击了省会奎达，中国建设的瓜达尔港，在几十个地点切断铁路和公路。这次冲突，俾路支各派武装有联合的趋势，巴基斯坦军队也不得不扩大了对俾路支平民的镇压，进一步扩大了游击队的群众基础。俾路支就在巴基斯坦和伊朗之间，基本保证伊朗从东方拿不到任何支持。

然后说北面。隔着里海是俄罗斯，在乌克兰已经打到第四年。春节前后，乌克兰借着俄罗斯被赶出星链体系的机会发动反击，虽然只抢回几百平方公里，但俄罗斯肯定是没有支援伊朗的力量。

然后是阿塞拜疆和亚美尼亚。前两年的纳卡战争，阿塞拜疆完胜，不仅夺回了纳卡地区，还强迫亚美尼亚签订条约，提供一条穿越国土的交通走廊使用权，连接两块飞地。从交通角度说，伊朗北部只有阿塞拜疆一个国家。

阿塞拜疆的民族和伊朗相似，但是最近几十年的背后支持者是伊朗西北邻国土耳其。而土耳其一向和伊朗有竞争关系，最近虽然有所缓和，但也不可能在关键问题上支持伊朗。2025年，在库尔德问题上，美国默许了土耳其的扩张，换到什么承诺不清楚，反正土耳其听说伊朗国内动荡，第一反应是加强边界上的高墙防线，挡住难民。

然后是西面，伊拉克有大量的什叶派，伊朗插手不少。前几天伊朗人民暴动，伊朗就用了从伊拉克撤回来的什叶派民兵镇压。但是，如果美国舰队在南面波斯湾行动，伊拉克和伊朗一样，都会变成内陆国，除了简单的人力之外，提供不了什么资源。在伊拉克背后，是土耳其支持的叙利亚。是美国大规模驻军的约旦，再加上最讨厌伊朗的沙特。所以这个方向也没机会。

此外，伊朗最强的战略威慑，就是封锁南面霍尔木兹海峡，封闭波斯湾，希望能给全世界带来石油危机，尤其影响美国的物价。但是两个月之前，美国刚刚拿下了委内瑞拉，快速增加了石油出口。美国开始轰炸伊朗之后，欧佩克组织讨论增产计划。2025年美国是石油出口国，生产的石油有30%要出口。伊朗对石油市场的影响，很难说有多大，对美国尤其不明显。

可以说，美国和以色列打伊朗，选择了一个不错的时间点，周边国家不可能支援，其他大国也很难有间接插手机会。至于这个时间点是选出来的，还是创造出来的，现在无法判断，反正莫迪访问以色列，巴基斯坦打阿富汗，俾路支武装加强进攻，这几个时间点实在离得太近了。

前面列举这么多周边的变化，我并不是说特朗普或者内塔尼亚胡深谋远虑，在发起攻击之前就已经锁定了胜利。如果特朗普真的能操控一切，他不会这么晚才出动航母，错过了1月份伊朗的全国暴动。只是大国拥有发动战争的主动权，就算错过了最好机会，也可以选择相对合适的时机发动进攻。如果在政治方面得不到足够的承诺，不打也是一种选择。当年张召忠说普京有大智慧，不会随便吞并克里米亚，就是这个意思。

而伊朗，或者说其他中小型国家，不能决定战争开始的时间，只能假设大国一定会选择相对有利的机会进攻，唯一的应对，就是平时在战略上下功夫，多交朋友，积累物资，避免被大国突袭亡国。只要能顶住第一波进攻，身后还有盟友，后续就有办法。

这方面最好的例子是2022年俄乌战争。乌克兰没有亡国，直接原因是泽连斯基个人足够英勇，给基辅市民发枪，鼓励几万民兵顶住了俄罗斯空降旅。但乌克兰更重要的底气，是前任总统波罗申科重建了乌克兰军队，他一方面吸收苏联体系的优点，储备足够的后备动员力量，另一方面引进北约武器，发展信息化战争能力。在俄罗斯偷袭之前，乌克兰已经拥有和俄罗斯正面交火的能力。

所以，在2022年春天，俄罗斯进攻力量最强的时候，也只是借助白俄罗斯领土从侧面偷袭了基辅，正面的部队始终不能突破东部几个州的战线，现在还在进行几公里范围内的拉锯战。所以在基辅巷战的同时，正面局势没有崩盘。等到基辅、哈尔科夫、赫尔松方面的部队缺乏后勤，退回战前位置，乌克兰战争就长期化了。

伊朗统治集团在和平时期的紧迫感似乎不强，或者说因为内部矛盾太深，没有能力进行高强度备战。可能有的观众已经注意到了，刚才我逆时针介绍了伊朗的邻国，唯独跳过了里海东岸的土库曼斯坦。

这首先因为土库曼斯坦统治者的政策是闭关锁国，除了卖点石油之外什么都不关心。1995年土库曼斯坦和联合国达成协议，成为二战后第一个绝对中立国。想说服土库曼给自己当可靠后方，没那么容易。

当然，在发现被包围之后，伊朗还是努力争取土库曼的合作。土库曼继承了苏联的铁路宽轨，伊朗和中国一样是标准轨。根据德黑兰时报报道，2月27日下午——也就是美国飞机轰炸之前24小时，双方谈定了最新技术问题，希望能把每年的过境货运量从160万吨提高到400万吨。

但远水不解近渴，甚至我们不知道，伊朗统治集团是不是在以色列直接轰炸之后，才开始考虑后方建设。是不是发现美国的谈判条件无法接受之后，才愿意对邻国给出更多的让步。所以，不管战术上打成什么样，伊朗统治集团短期内只能靠自己对付美国和以色列，这是为平时的懈怠，也为内部的矛盾付出代价。
