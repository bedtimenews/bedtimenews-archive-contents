# 

大家好，2026年3月15日星期日，欢迎收看1029期睡前消息。请静静介绍话题。

<font color = "indigo">今年年初，中国科学院官网发布信息，调整组织机构。其中最让人关注的变化，是地方分院调整。中科院本来下设了11个分院，现在精简到9个，长春分院和南京分院都被撤销，职能分别并入沈阳分院和上海分院。

最近几年，中央层面一直在收紧财政支出，中科院这次缩减分支机构，是不是也承受了预算压力？</font>

![](https://mmbiz.qpic.cn/sz_mmbiz_gif/VZDFliaW3RYxVwThqFiaF4WevjWZnGsjUGbRQsWpiaFVMCwhoLBfzrBHfEIv0fSmOY0mmwO1Wic8exlvkpD8iapjaVpu0vJjY5hKTbA0wvORIpso/640?wx_fmt=gif&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=1)

跟钱的关系应该不大。虽然中科院还没有发布今年的预算报告，但是我们可以看到2025年的预算是1940亿，比前一年少34亿。这次调整，甚至也不能说明南京、长春的科研地位在弱化，至少南京的地位没有相对下降。去年11月，中科院刚刚在南京成立了工业人工智能研究所。从2010年到现在，中科院只有3个新建的研究所，南京这个新研究所，也是智能制造领域第一个国家级科研机构。



中科院现在合并分院，最有可能的原因，还是在科研模式转型期间，没有处理好自己的定位，不仅是分院，就是北京总部的行政机构，都找不到自己的定位。



这里要说明一下，中科院的分院，并不是主要下属机构。分散在全国的110多个研究所，才是大多数中科院人员的就业单位。这些研究所直属中科院总部，就算地方分院跟研究所在同一个城市，分院也不能指挥研究所，最多用自己的行政身份，帮研究所对接当地政府和高校。另外，中科院的分院也没有权限新建或者撤销实验室，你问各地的分院院长到底负责什么，权力边界在哪里，他自己也说不清楚。



科学院，从字面意思来说，是科学家聚会的地方。聚会形成规律之后，有了自己的聚会资格认证，得到认证的人就是院士。世界上第一个现代科学学会是英国皇家学会，大多数国家或者地区，都仿照英国模式，建立科学院或者学会。



作为社会团体，科学院的主要职能，首先是满足院士之间的交流，尤其是跨学科交流。其次是代表所有科研机构和院士，共同向社会争取资源。如果有一些基础研究很难拉到投资，科学院也会出一些钱，直接刺激科学进步。



虽然科学院很乐意跟政府拉近关系，从政府手里拿钱，给政府充当科技顾问，但在性质上来说还是自治团体。很少有科学院可以直接制订科研规划，具体管理重大项目。简单来说，大多数国家的科学院，对内是院士的社交平台，对外是院士代表的咨询机构，并不是政府机关。



中科院的情况正好相反。新中国第一个成立的科研管理机构就是中科院，而不是科技部的前身国家科委。1949年11月，中国科学院直接作为政务院直属部门，管理全国科研工作。国家科委成立要等到1956年。在这之前，中科院就是现在的国家科技部，管理拨款和人员。



国家科委成立以后，拿走了中科院的行政权力，但当时主要领导基本都是军队干部，没有多少科学家。中科院还是集中了全国所有科学家，国家科委要制定技术政策，必须要找中科院帮忙。比如说国家当年出台《十二年科技规划》，确定未来十年的科技任务，包括研发原子弹和计算机。这份文件名义上由国家科委制定，实际编制文件的人，还是来自中科院。五六十年代，科委和科学院并行的时代，科学院的定位，确实就是一个有权威性的实体咨询服务机构。



60年代末，中央政府担心跟苏联开战，为了提高决策效率，毛泽东要求精简机构。国务院将近90个部门都经历了重整，变成20多个。国家科委也被撤销，业务和人员又交回中科院。直到1977年，国家科委才恢复编制，在这期间，科学院对应今天的科技部，科学院分院，可以类比今天的科技厅。



建国前几十年重视中科院，完全可以理解。中科院1949年成立，前身是民国政府的中央研究院和北平研究院。当时全国受过高等教育的科研人员，还不到5万，有资格称作科研工作者的，也就几百号人。除了这几百号人，大学的主要任务是教学；企业的主要任务是复制外国技术，都几乎没有原创科研能力。所以最开始几年，中科院的主要任务是找人，通过李四光、华罗庚这些有海外留学背景的科学家，联络海外华人学者，把人集中到科学院体系内，尽快形成基本的科研能力。



新中国科研缺人也缺钱，所以在1953年，钱三强带领中科院专家代表团访问苏联，想要学习一些先进经验。钱三强先后参观了苏联科学院将近一百个研究机构，回国就给中央打报告，表示苏联把科学院当成科研“指挥部”，集中力量办大事，很适合中国的落后现状。



中央接受了钱三强的建议，把人才、设备和资金都集中到中科院，成立直属研究所，很快取得一些技术突破。比如说中国第一台通用数字电子计算机，是中科院计算技术研究所的成果。中国首次实现人工全合成结晶牛胰岛素，是中科院上海生物化学研究所的成果。为了研究原子弹，中科院近代物理研究所直接挖空了清华、北大的物理系骨干。



拿到几乎所有人才和资金之后，中科院既可以制定国家科研计划，又能够直接执行计划，自然就成了全国科研主管部门。前面提到，中科院为了集中力量办大事，成立了一批研究所，但这些研究所，并不是都在北京。比如中科院在上海有生物化学研究所，同时在武汉建了水生生物研究所，长春有应用物理研究所。下属机构遍布全国，怎么管理就成了一个现实问题。



最直接的问题，就是通讯。60年代从北京到中国基建最发达的哈尔滨，坐火车也要一天一夜。如果打长途电话，大城市之间只有几条线路，需要接线员在交换机上手动操作，为了避免占线，就算领导干部，打电话也要提前预约。另一方面，当时全国各大城市资源都比较紧张，中央科研机构并不一定直接促进地方经济发展。研究所要说服地方政府，拿到土地和粮食指标，也是个难题。研究人员不擅长这些行政问题，所以中科院总部决定，在各大城市设立行政机构，代表当地所有研究所，和地方政府交涉。



![](https://mmbiz.qpic.cn/mmbiz_png/VZDFliaW3RYys6k8cCu4lrZw4jgqA2cU71teL9Y2n17icUbFjcp8yJXW9R7jia2X2drrD3SicSzSodia7sico8XpYap28ia2fxciaezrKyOxa8Wv3Ms/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=2)


1952年，中科院成立了第一个分院，东北分院。后来中科院的研究所越办越多，分院数量也不断增加，最多的时候达到26个。到了60年代，中科院发现分院设置太多，又开始缩减数量。按照中央划分的六大行政区，比如说东北、华北、华东，各设立一个分院，再加上一个新疆分院。新疆分院的存在，说明通讯和交通障碍是设置分院的重要理由。



70年代，中科院虽然没有被裁撤，但是也经历了精简，大量研究所和地方分院都被划给地方政府。1977年中科院恢复了分院体制，各地的研究所又回到中科院体系，由分院管理，分院终于成为了实权机构。



中科院分院现在的格局，是上世纪80年代确立的。按照50年代的逻辑，有些分院设置显然不太合理。比如说这次合并的长春跟沈阳，南京跟上海，都是邻居，也都在同一个大区，地盘和责任划分都不明确。可能是计划经济时代，重点工业城市的发言权比较大，被迫平衡一下。



但是中科院分院的好日子没过几年，就发现自己的地位快速下降。因为经过几十年发展，中国培养了大量科技人才，中科院已经装不下了。大量人才流向高校和地方国企，中科院没法再垄断科研人力资源。另一方面，国家也在逐渐调整计划经济体系，财政分配比例开始向地方倾斜，中科院系统也没法再垄断经费。



从80年代后期开始，各地大学变成了科研主力，各个城市都想培养自己的清华北大，跟中科院在同一个领域竞争。进入新世纪后，中科院又多了一个竞争对手，民营企业的研发部。比如说华为、比亚迪和宁德时代，在主业方向的研发投入，都超过了政府。对于21世纪的中国社会来说，搞科研，就像吃饭喝水一样正常，不需要一个专门的机构来管理每个单位吃什么。

![](https://mmbiz.qpic.cn/mmbiz_png/VZDFliaW3RYxoo0LicUUZQwEOTlxd7OuHokcssh4AKuZVaWiagaZiaV2iaLicFBibL1N9F9xcicZpriakLPG0T14MRYPWxI8hY0Aia5icIGViacCiaDCKic0Q/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=3)


更大的问题，是中科院也管不住自己下面的研究所了。1985年，中央发布了《关于科学技术体制改革的决定》，核心要求有两个：



第一，国家减少给研究机构的拨款，但是研究机构可以承包国家项目，给企业提供咨询服务，帮企业做委托研究，自己挣钱。机构还可以跟企业合资，一起开公司。研究人员只要不影响工作，也可以在外面创业，自己挣钱。这份文件同时确定了一个目标，就是三到五年之内，研究机构要基本做到自己养活自己。



第二个变化，是所长负责制。所长只要不违法，可以自己决定组织和人事。如果所长找到门路，不靠拨款就可以养活自己，还可以用赚到的钱给员工涨工资、发奖金。



研究所不从中科院总部拿钱，也不需要中科院批准人事安排，实际上就是独立了。连总部都管不到下面研究所，地方分院更做不到。到这个时候，中科院研究所跟总部的关系，更像共用一个连锁品牌的加盟商。分院作为分公司，肯定管不了和总部有合同的加盟机构。



1998年，国家科学技术委员会正式改名科技部，拿到了科技事务的行政管理权，负责组织攻坚项目，分配科研资金。中科院的权限进一步被削弱，变成了一个单纯的学术机构。理论上来说，中科院就算把分院全都砍掉，也不会有什么影响。



<font color = "indigo">既然中科院已经没有太大影响力，还有存在必要吗？是不是应该像英美科学院一样，变成一个纯粹的荣誉机构？</font>



各地分院没什么存在必要，但是中科院总部，可以往更有用的方向转型，或者说回到最初的科学院定位，负责三个方面：院士交流，科学咨询，和基础研究的关键投资。



在基础研究方面，德国有不错的的模板，马斯克.普朗克科学促进会，以及亥姆霍兹联合会。其中马普所负责基础研究，亥姆霍兹联合会重点是前沿探索。两个协会每年固定从德国政府拿钱，承担国家科学任务，已经持续了一百多年。



协会不干预下属研究所的日常科研，但是可以协调一些急缺经费，提供战略规划。德国连续打输了两次世界大战，都能快速恢复，离不开这套科学体系。



当然了，德国最近二十年在科研领域没有进步，几乎完全错过了IT技术革命，科研方面成果不突出。但这是德国政府和企业共同的选择，不能把所有责任都丢给马普所。起码对比美英科学院，马普所的积极作用要大得多。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/VZDFliaW3RYzZ3a3D9DS6smLx6PzFUAG6styYcxa8GezNFXjibh3E2SXWpkLHtoCqN8gOS5iaXxSB0sGBZfCQUptlc4ticgrQHfia4T3ibKXbqpzk/640?wx_fmt=jpeg&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=4)


除此之外，科学院的功能就是以院士为中心，促进跨学科的社交，也向科技部和其他国家机构提供非强制性的咨询服务。这方面的工作，可以通过院士自治来完成。相信以科学院院士的地位和资源，不需要国家特别安排行政级别，也能组织中国最高级的科研俱乐部。



因为几十年的惯性，可能有人会觉得撤销科学院分院，乃至撤销行政化的科学院总部，有点不可思议。但只要对比一下工程院就知道，科学院的现状才是一个畸形状态。



中国工程院从来没有分院，其次也没有中央行政机构。打开工程院自己的网站，可以看到，一切权力来自院士大会，其次是授权的主席团。工程院所属的机构，除了几个学术委员会之外，行政机关被精简到最小规模。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/VZDFliaW3RYzxxJb3VXux7tINwyRVx04ZDRBySGXtn38ro5LIwpc4w9hCklcrPNDVEEwonjoVOZoc2abhBl1zV1YXYXicW9Uk34Ao2TNjwzcU/640?wx_fmt=jpeg&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=5)

工程院网站列出了五个工作重点：



战略咨询，学术引领，科技合作，人才培养，对外交流。


![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/VZDFliaW3RYw636QGU9Oia57XbWy0tibSW76GRXIQFz4ibEZY8ceibhg3TwsbjqF8qDsIHcJCReKFgh8P1LEfATcUnkT37OR1d73TWzKdN8ycyvw/640?wx_fmt=jpeg&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=6)



其中人才培养项目，重点就是维护一个光华奖的权威性。学术引领项目，重点是关键研究成果的出版。这显然不占用什么资源。



所以工程院的日常业务，就是剩下的三项。战略咨询，科技合作，对外交流。具体来说，就是为院士的学术社交提供服务，组织院士联合起来回答全社会的重大疑问。

![](https://mmbiz.qpic.cn/mmbiz_jpg/VZDFliaW3RYwaTQdfoStEzbRprr5UkFvQkxfic3rsIJA6zTIT3E2nIgcicYjX7N6vEYGu9FdhxkOJ0a4DHEWL5S2uQmSicJd62GHJaKe4WvGOao/640?wx_fmt=jpeg&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=7)

![](https://mmbiz.qpic.cn/mmbiz_jpg/VZDFliaW3RYxnMa8Sgniad53KibCEBdIs5K2z0jkOKbvWqETJHTYK3x1cJCBnRSAVZw2NTrkSgHFrRuITiaECsGjpYFgl88Nd977JJ6pMDH6qiag/640?wx_fmt=jpeg&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=8)


科学院有庞大的行政机构，直接管理十几个分院和上百个研究所。而工程院就能精兵简政，专心服务院士，原因还是前面说的。20世纪的中国太落后，科学研究资源太稀缺，所以必须集中到一个实体机构下面来搞科研。同时，搞工程应用技术的门槛比较低，企业和普通的学校都能做，集中到一个机构来管，反而降低效率，影响技术成果的转化。所以工程院不需要也不可能得到全国性的行政权力。



到了21世纪，科学研究资源已经不稀缺了，至少比20世纪的工程研究资源要普及。企业和地方的研究所，都有能力搞科学研究，也有动力进行科学研究。所以，正确的改革方向，就是科学院学习工程院，让研究所和大学独立运行，总部变成咨询机构和院士俱乐部。



中科院显然也感受到了新形势的压力，所以主动解散一部分地方分院，把编制留到总部，想做点事情。这个改良方向是没错的，错在让中科院自己决定怎么改。众所周知，一个花拨款的行政机构，不可能给自己做截肢手术，只能论证自己改良之后长期存在的合理性，争取更多的拨款和权力。



比如说，在中国之外，只有少数几个停滞的前苏联国家，才能看到由科学院直接管理的大学。中科院本来就管理合肥的中国科学技术大学，在科研订单和科研资源扩散之后，中科大也和中科院的其他研究所一样，有了独立性。对于这种趋势，中科院的对策不是鼓励大学独立发展，而是把自己能控制的少数研究所资源集中起来，在北京昌平又搞了一个中国科学院大学，简称国科大。



现在就连中科院自己也说不清两个大学之间的关系，唯一的解释，就是官僚机构不甘心权力流失，达成了集体默契，要把权力抓回来。所以，中科院的改革，不能指望他们自己刀刃向内，不能调整几个分院就过关，必须由全社会来讨论中科院的定位和存在价值。



最后说一句，中科院如果一定要讲中国特色，要维持建国初期的优秀传统，我倒是有一个建议，就是让社会科学重新变成科学。中科院第一任院长是擅长文学和历史研究的郭沫若。他从1949年当院长，一直到1978年，正好覆盖了中科院相对地位最高的年代。这期间，中国没有社会科学研究院，大多数对应的研究，都放在中科院的哲学社会学部。比如说历史研究所、世界历史研究所、世界经济研究所，都是中科院来管。

现在中国社会学研究的现状，大家都知道，不用我多说。其中一个原因，就是普通人甚至专业研究人员，都不认为社会科学是科学，所以可以立场先行，不接受现实的批判和验证。



同时，现实的研究证明，社会科学不仅是科学，而且可以和自然科学放在一起研究，相互贯通，相互促进。比如说2021年睡前消息[361期](/main/301-400/361)的标题是《复旦新校长，生物院士用语言学研究历史》。我们一起回顾当时的评论：



> 无论是语言学还是生物学，都曾经是模糊的学科，研究者的主要工作是搜集样本，而不是研究样本之间的联系。但是到了21世纪，随着数学工具和计算机工具的进步，语言和生物都可以分解成最基本的特征要素，用统计手段来追溯发展脉络。历史记载甚至是神话传说都可以用数据来验证。人类期待了几百年的“唯物主义历史学”终于可以落地了。这就是科学一步步取代玄学的过程。

5年时间过去了，AI技术降低了各学科之间相互借鉴的门槛，中国社会科学相对落后的问题也越来越严重。所以我说，中科院如果一定要保留一点中国特色，和欧美的院士俱乐部保持区别，我建议可以把社科院并回来，建立一个更大的科学俱乐部，让中国的社会科学更像科学。
