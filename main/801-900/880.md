大家好，2025年3月25日星期二，欢迎收看880期睡前消息。请静静介绍话题。

<font color = "indigo">3月11号，巴基斯坦俾路支省发生劫持列车事件，武装人员攻击了巴基斯坦军警和家属的通勤列车，扣押450名人质。当地分离主义组织“俾路支解放军”宣布对此负责，要求巴政府释放其被捕的成员。

事发之后，俾路支解放军释放了妇女儿童人质，继续扣押剩下的军警和公务员。巴基斯坦军队攻击列车，最后至少70人死亡，是近年巴基斯坦伤亡最惨重的袭击。

几天后，俾路支地区又有武装分子发动自杀式袭击，造成多人死亡。过去几年，俾路支地区治安动荡已经是常态。中国当地企业也多次受到袭击，出现人员伤亡。

俾路支地区出了什么问题？</font>

**先说结论。俾路支分离主义组织发动袭击，并不专门针对中国人，也不针对其他任何具体国家，而是要打击所有在俾路支地区开展投资，或者勘察资源、承包工程的外人。就算是巴基斯坦自己的企业，只要不是俾路支人办的，也要打。因为他们的目的就是破坏俾路支地区的建设。巴基斯坦国家机器不关心俾路支人的生活，他们也不允许国家机器从俾路支获得收益，最好能让俾路支成为巴基斯坦的财政黑洞，社会伤口。**

俾路支省有35万平方公里，接近中国的云南省，是巴基斯坦面积最大的省份，占领土的44%。但俾路支的人口是是全国最少，2023年有1490万人，略多于天津市，只占巴基斯坦人口的6%。

和中国的边疆省份类似，俾路支省内的族群分布很复杂。粗略统计，以说俾路支语为标准，广义的俾路支人，只占全省人口的四成。以普什图语为母语的人口也超过了三分之一，两者差距并不大。另外，在俾路支省，人口最密集，经济最发达的省会奎达及周边地区，恰好是普什图系的人口优势更大。

本来俾路支地区在巴基斯坦影响力就最小，内部的复杂情况又进一步造成了力量分散，所以俾路支人的待遇进一步边缘化，在资源分配方面长期受歧视。俾路支的gdp和人均gdp都是巴基斯坦全国最低水平。

除了穷，俾路支的所有社会指标几乎都排在最后。它是全国唯一识字率不到一半的地区，人均收入、预期寿命、医疗机构密度、有供水的家庭比例、都大幅低于全国平均水平。

俾路支省以干旱山区为主，的确不适合发展经济。但在俾路支人看来，本地山区有煤矿、金矿、铜矿，尤其是天然气丰富，还有自己的出海口，1000多万人过不上好日子，原因是巴基斯坦主流社会歧视自己。尤其是伊斯兰堡的统治者不拿俾路支人当公民，只开发资源不在本地搞建设。所以分离主义组织得到了基层支持，其中就包括这次发动袭击的“俾路支解放军”。

巴基斯坦政府是一个高度军事化的机构，不是军人集团直接执政，就是军方在后台操控。所以，面对独立诉求，很自然地选择了派兵镇压。但镇压效果并不好，双方的暴力行为循环上升，本地武装规模越来越大，袭击越来越多。

2023年，俾路支武装攻击瓜达尔港，炸了有中国工程师的车队。2024年，叛军一度占领瓜达尔港港务大楼。虽然袭击最后被打败，但敢于直接进攻军警重兵保卫的关键地点，说明叛军已经有相当程度的人力储备，以及很清晰的战略目标。

今年1月，俾路支分离主义武装首次攻占了一座巴基斯坦的城镇，停留了24小时。从过去几年的趋势看，俾路支解放军即将从破坏组织发展到半公开的叛军。

<font color = "indigo">巴基斯坦政府真的歧视俾路支地区吗？</font>

**的确有不公平对待。但更准确的说法，是旁遮普和信德两个大省的封建势力垄断了经济资源，绑架了本地人民，客观上达成了歧视俾路支和其他边缘族群的效果。**

历史上并没有巴基斯坦国家。1947年英国退出南亚殖民地，以穆斯林为主的地区自然地组成了巴基斯坦。国名直译过来就是“圣地”，说明整个国家只有伊斯兰教可以作为文化共识，后来又加上了和印度对抗。

成立国家之后，因为各地区、各族群差异巨大，巴基斯坦基本上按照语言和民族界线划分了省份，导致“省籍”认同深入人心。接下来几十年，上层统治者，无论是大地主还是军官，都喜欢按照“省籍”分配利益，提升官员，这进一步强化了核心省份的封建秩序，也加深了各省之间的隔阂。当边缘族群表示不满的时候，巴基斯坦政府和军队为了保证自己的地位，往往会利用族群歧视获取支持，进行野蛮镇压。

[第376期](/main/301-400/376)节目介绍过，最初的巴基斯坦还包括现在的孟加拉国，当时被称为东巴基斯坦，提供了整个国家大多数出口商品。因为孟加拉人口不满意西巴基斯坦占有多数外汇，巴基斯坦军方就在东部制造了二战之后最恐怖的直接屠杀，直接杀死上百万人，烧掉了上千万人的房子。而且还刻意挑出孟加拉地区的大多数受教育人口，优先屠杀，想在人口和文化两方面消灭国内第一大族群。

现在，巴基斯坦军事封建集团已经退回了西部印度河流域，但治理国家的水平，并不明显超过20世纪的水平。俾路支人看到本地的天然气被开采，港口被开发，但生活水平没有提升，甚至没有多少就业机会，认为自己被歧视，也是很自然的结论。

<font color = "indigo">为什么巴基斯坦各省会形成封建化的族群？</font>

印度洋北部的平均水温达到25度，是世界上最热的开放性海洋，提供了最充沛的水汽；喜马拉雅山是世界上最高的山脉，扰乱了附近的风向，让一部分水汽深入内陆。南亚地区夹在两个极端地形之间，接受低纬度的强烈阳光照射，形成了多样化的气候。在相对平坦的地形上，季风农业区过渡到荒漠地区，可能只需要100公里的距离。

所以，南亚地区形成了复杂的社会结构，各族群长期混居但不融合。在已知的印度文明形成之前，南亚就已经形成了牢固的身份世袭制和族群内婚制。后来的印度教文明在既有的基础上发展了种姓制度。甚至伊斯兰教到了印度，也要和种姓制度结合。对于地方统治者来说，带领大族群欺负小族群，是天经地义的权利，也是自己巩固现有地位的方式。

1947年印巴分治，英属印度一分为二，边界穿过了英属印度时代的两个超级大“省”，旁遮普和孟加拉。东孟加拉就是曾经的 “东巴基斯坦”，独立后的孟加拉国。

而西旁遮普是西巴基斯坦的核心，巴基斯坦的国语乌尔都语也来自旁遮普地区。东巴基斯坦独立后，旁遮普的人口占全国的44%，经济资源超过全国一半。当地上层社会占据了最多的经济资源和军政职务。

同时，信德省包含巴基斯坦最大城市，最大海港卡拉奇，在进出口贸易、工商业方面占据优势。拥有仅次于旁遮普集团的发言权。以信德省为主要根据地的布托家族和人民党，几度上台控制中央政府。现任总统就是布托家族的家主扎尔达里。

巴基斯坦第三大族是普什图人，聚居在西北部山区，在中央政府的影响力比较弱。但普什图部族历史上就占据了南亚和中亚之间的山谷通道，协助数不清的中亚统治者征服过印度，自己也形成了强悍的军事文化。而且普什图族是跨境民族，在邻国阿富汗也是第一大族。本族的力量可以跨境得到支援，也能退到边境另一侧得到保护，不能被中央军队轻易压服。

所以，无论是英属印度政府，还是现在的巴基斯坦政府，都把普什图地区看做需要安抚的山区硬汉，转移了不少经济资源。巴基斯坦城市主流社会一般也不公开歧视普什图人。现在巴国内民望最高的政治人物，前总理伊姆兰.汗，就是开伯尔省出身的普什图人。

在正式的省级单位之外，巴基斯坦还有一个巴控克什米尔地区。因为巴基斯坦与印度长期争夺主权，克什米尔也一直得到不少额外支持。

数了一圈，只剩下俾路支人的处境最尴尬。虽然俾路支人也算是跨境民族，但在另两个邻国伊朗和阿富汗，俾路支人也是边缘族群，发不出多少声音。尤其是伊朗俾路支人不是伊朗主流的什叶派，与政府关系很差，当地也有分离主义活动，伊朗政府经常要镇压。

所以，在半公开歧视弱者的南亚社会环境中，俾路支人是是力量最弱，外援最少的巴基斯坦群体。不仅被统治者歧视，甚至也被巴基斯坦主流民众歧视。

![880-6.jpg](https://img.bedtime.news/2025/04/02/67ed5d230cc92.png)

几十年来，巴基斯坦从文官政府换成军政府，从军政府又换成人民党、穆盟轮流执政，但没人关心俾路支人的诉求。平时统治集团假装忘掉有这么一块地方，遇到反抗，就毫不犹豫地发起镇压。

<font color = "indigo">镇压了几十年，本地的反抗运动越来越强，巴基斯坦统治者有没有考虑投入一些经济资源，解决俾路支人的怨气？</font>

解决分裂问题最好的方法，永远是用军事力量做后盾，但不轻易使用暴力。然后长期向本地居民，尤其是愿意配合中央政策的居民投入经济资源，就能软化独立情绪，最终破坏分离主义的动员能力。但是，过去几十年，巴基斯坦浪费了太多的发展机会，也制造了太多的族群矛盾，现在就算想回头解决问题，也很难。

这里我想回顾一下[556期](/main/501-600/556)节目对巴基斯坦的评价：

> *伊姆兰·汗下台之后的巴基斯坦，在政治和社会结构上已经是一条僵尸了，如果不经过一次彻底摧毁旧社会结构的大动荡，或者借着世界大战的机会，最后发挥自己的军事遗产，巴基斯坦恐怕会彻底失去现代化的机会。*

具体解释一下，巴基斯坦立国就是一个准农业社会，在失去孟加拉国的外贸能力之后，仅有的经济剩余都拿来对抗印度，维护阿富汗方向的边境，还嫌不够用，必须找到强大的境外援助才能生存。

过去一代人时间，巴基斯坦人口爆炸，现在已经达到了2.5亿。因为外国援助没有跟着人口同步增长，巴基斯坦的人均gdp停滞在1500美元左右，和2018年几乎一样，但购买力明显下降了。财政方面，政府外汇储备只有几十亿美元，外债倒是有1000多亿。

就在上个星期，巴基斯坦刚刚把世界上最大的铜金矿，雷科迪克铜金矿的控股权卖给加拿大巴里克公司，换取了第一期40亿美元注资。不意外的话，金矿的未来收益首先会被巴基斯坦政府拿去维持财政平衡。刚刚卖掉的铜金矿，恰好就在俾路支地区。这就是当前巴基斯坦基本矛盾的缩影。

![880-7.jpg](https://img.bedtime.news/2025/04/02/67ed5d230c936.png)

拥有2.5亿人口，还要把超级大矿卖掉换现金，核心原因是人口没有加入国际分工，不能变成廉价劳动力。2025年的今天，巴基斯坦一半以上的适龄人口没读过中学，女性识字率还不到一半。而孟加拉国在独立之后坚持几十年搞基础教育，发展出口导向性经济，人均gdp已经达到了2500多美元。而且孟加拉也有接近2亿人口，占住了所有可能的低端产业升级方向，让巴基斯坦想搞血汗工厂都没有起步机会。这可以看做东巴基斯坦对当年大屠杀的经济复仇。

经济上没出路，巴基斯坦除了卖矿，唯一的价值就是政治。再说的明白一点，就是在南亚地区牵制印度，避免印度控制整个次大陆。

巴基斯坦最大的援助来源并不是中国，而是逊尼派穆斯林占多数的中东石油富国，比如沙特、阿联酋、卡塔尔，多年累积，向巴基斯坦提供了几百亿美元的资金。在援助之外，中东石油富国还愿意花大价钱雇佣巴基斯坦的士兵，甚至高价购买二流武器，间接支持巴基斯坦核计划。

这肯定不是因为宗教精神，更实际的理由是中东国家也担心印度力量的增长。因为印度劳工实在太多了。19世纪到20世纪前期，英国围绕印度洋建设帝国体系，大量使用印度劳工，在印度制造了跨海打工的文化。现在沙特本国人口1900万，使用400万印度劳工。阿联酋本国人口130万，使用360万印度劳工。

这两个国家到印度领土的直线距离只有1300公里，相当于北京到南昌，中间隔着无遮无挡的阿拉伯海。如果没有巴基斯坦制约印度，中东国家很担心印度跨过阿拉伯海投放力量，甚至接管印度最缺的油田。

除了中东和中国之外，美国和欧洲虽然乐意和印度发展关系，也不希望看到一个完全控制南亚次大陆的印度，所以时不时给巴基斯坦一些资金，避免国家崩溃。

但所有这些支援，都是救急不救穷。巴基斯坦人口已经超过2.5亿了，生产方面的价值近乎为0，没有哪个国家愿意拉他们进现代社会。巴基斯坦统治集团要自救，最熟悉的方案还是开发俾路支山区的矿产，同时无视俾路支人的发展需求。既然俾路支人没有强大的国际援助，继续用暴力压制这个1000多万人口的小民族，可能还是眼下没有办法的办法。

俾路支人自己也知道不会有强大的国际援助，所以从一开始，就不指望正面打败中央政府。他们最近几年选择恐怖袭击方案，无差别攻击开发资源的企业。目的就是要“两败俱伤”，动摇中央政府的统治意志，换取一个半独立地位。上个月，俾路支武装再次袭击中国中冶山达克铜矿，这个项目也在俾路支省，当年被看做雷科迪克超级铜金矿的前置项目。现在既然雷科迪克矿卖给了加拿大公司，就轮到他们的工程师担心安全问题了。

<font color = "indigo">巴基斯坦就没有人想永久性解决俾路支问题吗？</font>

毕竟已经是21世纪了，巴基斯坦也有一部分人考虑过更进步的方案。比如说前总理伊姆兰.汗，对俾路支问题的态度就比较温和。在他执政期间，巴基斯坦政府第一次承认长期忽视俾路支省，承诺要投入更多资源。他甚至与分离主义组织进行过接触，想谈一个停火方案。

![880-8.jpg](https://img.bedtime.news/2025/04/02/67ed5d230ca94.png)

这也许与伊姆兰.汗出身于另一个少数族群有关。但更可能是因为，他和他身后的正义运动党，主要代表了城镇中产阶级和平民的利益。

城镇市民群体有很多缺点，但与前现代农业社会相比，这个群体还是更有进步性。其中一个优点，就是他们认为“身份”是可以变动的。所以，现代市民可以抛下小地域和血缘氏族，支持现代化的国家认同，避免大多数资源用于内耗。

伊姆兰.汗和正义运动党能上台，原因之一，就是大多数传统政党、政客无休止地炒作地域、族群话题。在经济停滞几十年的背景下，这让普通市民感到厌烦。而伊姆兰.汗作为少数族裔的普什图人，同时也是巴基斯坦“国球”板球最杰出的国家队队长。现在巴基斯坦的城市化率也超过1/3了，伊姆兰.汗可以代表超越地域和族群的国家形象。一部分市民希望伊姆兰.汗能超越传统，打破残余的封建社会结构，营造“巴基斯坦国族”概念，消灭内部族群矛盾。

现在，俾路支问题已经成为巴多数主流政党攻击对手的重要武器，越来越多政客都要表现自己的态度最“强硬”。另一方面，巴基斯坦军方连续多年插手阿富汗局势，自己也直接策划对印度的恐怖袭击，并承受报复。无论有没有俾路支武装，恐怖袭击在巴基斯坦都是日常生活的一部分。这反而提高了巴基斯坦普通人的耐受力，并不认为俾路支武装经常性打死几十个人是大事。所以，俾路支分离运动可能会长期停在他们预期的阶段——和巴基斯坦中央政府互相伤害，但无法达成任何妥协。

上周五，睡前消息的[878期](/main/801-900/878)视频现在已经下架了。这是开播6年以来，我们第一次主动永久删除内容，原因是出现了明显的事实错误。具体来说，我们根据自己的经验，草率推断2月份的广西百色溶洞科考是广西师范大学的项目，进一步认定相关单位隐蔽了自己的名字。

![880-10.jpg](https://img.bedtime.news/2025/04/02/67ed5d25a8dcb.png)

视频发出后，有人看到活动的组织者出面澄清，在微博上说

> *“我还没死呢……这博主就是别有用心！”*

可能观众会误以为我们在节目中把人说死了，这里要解释一下，878期的节目并没有说任何人在事故中伤亡。但我们还是要向广西师范大学表示道歉，其次也对影响了当事人的生活道歉。因为从目前的情况看，活动完全可能是几个人私下组织的，并不一定对应任何公立机构或企业项目。接下来，我回顾一下错误产生的原因，尤其要反思自己的“常识”偏差，以后要尽量避免类似问题。

首先，从新闻报道惯例来看，以往中国的科考救援报道，都能对应到负责单位。所以我们想当然，默认这次科考也必然有对应的负责单位。

但我们仔细查阅国家法规文件后，发现国内法律并没有要求科学考察必须由法人单位来组织，甚至没有明确定义过科学考察行为。所以，我们当时是抱着“寻找项目单位”的思路去搜索，证据越显著，结果错得越厉害。

对于野外科学考察活动，国内目前的限制政策，主要是针对自然保护区、野生动物、历史文物、外籍人员，以及国家秘密。

![880-11.jpg](https://img.bedtime.news/2025/04/02/67ed5d230dec3.png)

![880-12.jpg](https://img.bedtime.news/2025/04/02/67ed5d230c9b5.png)

在政府限制范围外，理论上说，科学考察没有任何限制。任何单位的内部规定，都管不到其他人。

至于说风险问题，私下组织科考，如果明确不存在雇佣关系，只要是成年人自愿参加，法律上的定性是“自甘风险行为”。即便过程中有人因为操作不当，造成了个人事故，其他人也没有责任。在社会越来越发达的21世纪，我们必须承认，一部分考察团可能就是没有对应单位。

878期节目原本的设定是“考察活动必须有单位。负责单位不应该在新闻场中消失”。在学习法律知识之后，我认识到，在信息不充分的前提下，不应随意质疑普通公民的冒险和科学考察活动。在此，首先向观众道歉，其次对广西师范大学道歉。欢迎各位观众保持监督力度。
