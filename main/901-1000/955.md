#

大家好，2025年9月21日星期日，欢迎收看955期睡前消息。请静静介绍话题。

<font color = "indigo">8月28日，欧盟委员会宣布，准备取消美国所有工业产品的关税，同时放宽美国农产品的准入条件。

督工，欧盟的这个决定，会对欧洲经济产生什么影响？ </font>

经济领域的实际影响不算大，政治方面的心理影响很大，彻底宣告了21世纪初以来，欧盟争取和美国平等地位的失败。美国再次证明，自己一个国家，对世界上中国之外的任何力量，都具备碾压式优势。



先说欧盟减税的问题。



过去几十年，欧盟和美国互相减免了很多产品关税，有三分之二的工业产品，早就是零关税了。剩下的三分之一，税率也不高。按照统计，今年上半年，欧盟队美国工业产品的平均税率，只有1.35%。税率最高的汽车，也只有10%。所以我说，欧盟的这个决定，对欧盟市场不会产生多大冲击。



反过来说，这次特朗普对全球发动贸易战之前，美国对欧盟工业品，大部分都是零关税，小部分产品就算征税，税率也低于5%。特朗普上台以后，税率提高到了15%。



现在欧盟主动让步，对美国工业品取消关税，但是美国并没有对等取消，而是继续征收15%的关税。此外，对钢铁、铝这些原材料，美国有权征收50%高额关税。同时，欧盟也跟韩国、日本一样，承诺要大量购买美国能源，在美国增加投资。换句话说，欧盟并没有因为政治团结，就争取到比日本、韩国更好的贸易地位。



贸易伙伴之间，公开实行不对等关税，一般只有两种情况。第一种是新加坡、香港这种城市经济体，只做转口贸易，所以主动取消自己的关税，不在乎别人加多高。另一种是发展中国家面对发达国家，在WTO的保护下，用高税率保护自身产业。



欧盟和美国，都属于高度发达经济体，过去不管加税或者减税，大家都会保持一样的税率，算是互惠交换。现在欧盟全面取消关税，美国却把关税提高了几倍，就算总税率不高，也证明了欧盟没有什么谈判筹码。



等到欧美正式落实协议内容，形成新的贸易格局，工业领域的投资人，就会优先考虑美国。所以未来很长一段时间，欧洲都很难推翻这套新架构，在工业领域跟美国的差距，只会越来越大。



<font color = "indigo">加关税是特朗普提出的新政策。换一个总统，美欧关税地位会变回去吗？</font>



就算换总统，欧盟的待遇也很难改变。现在的美国政治气氛，任何人想要取消贸易特权，都属于政治自杀。特朗普发动贸易战以来，欧盟这次让步，是美国赢得最多的一次，也是特朗普最成功的政治遗产。下一任总统只要想保持支持率，就不会随便放弃。



<font color = "indigo">有观点说，过去几十年的美欧贸易，欧洲一直处于顺差地位，占了美国很多便宜。现在欧盟同意美国单方面享有贸易特权，是为过去还债。

督工，你怎么评价这种观点？</font>



确实有经济理论支撑这种观点，叫作“两个平行市场”。



按照这套理论，不同国家之间的进出口贸易，只有保持基本平衡，对贸易双方才有利。如果出现单边贸易逆差，就有国家受到剥削。这套理论认为，要避免这种情况，国家应该制订长期贸易协定，消灭自由贸易带来的不平衡性。



这套理论虽然有一定道理，但发明人是约瑟夫斯大林，主要运用的区域，是苏联和东欧社会主义国家。西方的主流经济学观点完全相反，认为只要外汇支付能力没问题，贸易逆差完全可以接受。



但就算欧盟采取了斯大林的理论，也没法解释这个一边倒的结果。

 

以2024年美欧贸易为例，欧盟的货物贸易，确实有2000亿欧元的顺差。但是同一年，美国对欧洲的服务贸易，也有1480亿的顺差。把两个数据合并，整个2024年，欧盟对美贸易顺差只有500亿，对比两国高达1.68万亿的贸易总量，占比还不到3%。在国际贸易市场，这个比例的定性是“基本平衡”。



就算是500亿的顺差，也经不起细看。欧盟对美国的服务贸易出口，很多是美国企业为了避税，进行的财务操作。爱尔兰就是一个典型例子。按照爱尔兰法律，如果一家公司总部在海外，只需要缴纳12.5%的企业所得税。这个税率，比美国低了3倍。所以苹果在爱尔兰注册了运营公司，所有美国以外的收入，都从爱尔兰走账，再从爱尔兰给美国总部支付专利费。爱尔兰每年对美出口金额达到2700亿，很大一部分，都是类似苹果的账务往来。



美国企业在欧洲避税，产生的利润和就业，最后还是会流向美国。说欧盟从对美顺差占了便宜，没什么依据。



欧盟本来也不占便宜，这次选择对美国全面让步，唯一的理由，是美国优势太大了。



在过去，欧盟一直是美国的核心盟友，所以在公开场合，美国一般不会全力施压，维持双方的表面“平等”关系。但是等到特朗普打破默契，欧盟发现，自己虽然还是全球第二大经济体，但早就不是美国的对手。跟美国打经济战，可能会给欧盟带来毁灭性的结果。



从数据看，美国最穷的几个州，比如说密西西比、西弗吉尼亚，人均产值和收入，比英国伦敦以外的任何地区都高。整个欧盟也只有德国，能跟美国的穷州比一比收入。上一次出现类似情况，还是二战刚刚结束，整个欧洲都被打成了废墟。



以前面提到的爱尔兰为例，爱尔兰将近五分之一的GDP，由美国企业创造。美国只要不怕国内经济动荡，完全可以自行立法，禁止美国企业利用爱尔兰避税。这对爱尔兰造成的破坏，可能比一场战争还大。特朗普给自己塑造的形象，是“无法预测”。欧盟政客谁也不敢赌，特朗普不会真的开火。



<font color = "indigo">既然欧洲和美国差距这么大，为什么维持了几十年的平等地位？</font>



因为欧洲在修复二战的损失之后，经济总量和质量一度赶上了美国，直到21世纪，欧洲还被看做美国最有力的挑战者。



1945年二战结束，欧洲战后重建速度很快，到上世纪五六十年代，欧洲国家已经远远超过了战前水平。整个冷战时期，欧洲的经济发展速度，大多数时间高于美国。到了八十年代，美国和欧洲各有优势。美国的优势，集中在前沿高科技和军事领域。但是欧洲有更好的社会福利体系，阶级和种族矛盾也没那么尖锐。在汽车、奢侈品和教育产业，欧洲和美国也基本打平。



九十年代冷战刚刚结束，苏联解体，美国和欧洲都吃到了“和平红利”。但是最初十来年，美国实际上占到的经济便宜不多。因为美国要保证全球的海权，军事开支不能下降太多。



而欧洲过去承担的主要任务是直接对抗苏联陆军，所以在东欧国家主动加入欧盟和北约之后，欧洲国家可以大幅削减军事支出，进行财务投资，刺激了经济增长率，反而相对美国有更高的经济增长。



其他国家也看到了欧盟的优势，看好欧盟超越美国。上世纪九十年代，中国官方智库和主流媒体都认为，欧盟可以打破美国“一超独大”的局势，形成全球多极化格局，有利于中国趁机发展。2001年，清华大学国际问题研究所有篇文章，请静静读一下：



> <font color = "indigo">欧盟不仅是未来多极中的一员,而且是当前走向多极化的领头羊。虽然它面临的困难还很多,但我们对它的决心不应低估。翻开世界历史可以看出,全球国际关系长期以来是由欧洲几个大国所主导的，只是在二战之后才形成两极体制。
>
> 但这种局面显然已不可能长期维持下去。欧洲有雄厚的经济实力,要武装起来并非难事。历史上曾经主宰世界的“欧洲中心”各国当然不肯屈居人下。欧洲独立自强、摆脱对美依赖已是既定方针。</font>



1987年，邓小平会见荷兰首相，也把打破美苏霸权的希望，寄托在欧洲身上。我给大家读一下：

> 我们在分析国际状况时特别注意欧洲，欧洲是决定和平与战争的关键地区。我们希望有一个联合、强大、发展的欧洲。如果下一个世纪五十年，第三世界包括中国有一个可喜的发展，整个欧洲有一个可喜的发展，那个时候可以真正消除战争的危险。

当时的中国，最高层领导和普通研究人员，都看好欧洲的发展前景，原因是产业层次还看不透美国的优势。20世纪末，IT和早期互联网技术革命刚刚开始，传统产业还占据主导地位。欧洲是工业革命的发源地，在技术和品牌两方面，都有深厚积累。



就拿家用汽车来说，七十年代石油危机以后，美国汽车业竞争力一路下滑。高档汽车打不过欧洲，廉价汽车打不过日本。冷战结束后，全球经济高速增长，欧洲和日本的汽车品牌分到了最大一块红利。比如中国普通人喜欢桑塔纳，有钱人买丰田奔驰。



同样的情况，也发生在机械、化工行业。当时中国企业急着扩张产能，买设备的时候，采购欧洲和日本货的比例，远远高于美国。中国人比较美国和欧洲的时候，很容易更重视欧洲。



在社会和政治领域，欧盟对自身体制也非常自信。欧洲的绝大多数国家，比如说英国跟法国，彼此都有几百年恩怨。在欧洲政治精英看来，欧盟能够让这些世仇国家坐下来，变成伙伴，代表了“世界文明进步的方向”。从九十年代后期开始，欧盟开始快速扩张，十多年吸收了15个国家，也确实用规则而不是强权，保障了欧洲小国的权利。欧盟的快速发展，让很多欧洲精英相信，下一步就可以实现“欧洲合众国”。



经济和政治实力同时扩张，当时的欧盟跟美国打交道，就非常强硬。上世纪70年代，美国施行出口补贴政策，对所有出口商品减免50%税收。在欧盟看来，美国的行为属于不正当竞争，损害了欧洲企业的利益。此后二十多年，欧盟跟美国一直打国际官司，世贸组织不断裁定美国的补贴政策违法，美国又不断推出新的补贴法案。



2004年，欧盟不再跟美国打诉讼仗，直接动手制裁，对价值40亿美元的美国商品征收惩罚性关税。到美国2005年认输为止，欧盟对美国的关税达到15%。这也是世贸组织成立以来，规模最大的贸易制裁。



2007年，因为美国经历了911事件和第一次互联网经济泡沫，无论按哪一种统计方式，欧盟GDP都超过了美国，全世界都以为21世纪是欧盟的世纪。



但欧盟的好日子也就到此为止了。2008年美国次贷危机，第一个被引爆的全球经济体就是欧盟。从希腊开始，多个欧盟国家接连爆发债务危机，严重打击欧洲经济。从2007年开始，欧盟的世界冠军只拿了不到两年，经济体量就被美国重新反超，差距越来越大。现在欧盟4.5亿人口，比美国多了一亿。但欧盟的gdp总量只相当于美国的2/3，已经和中国不相上下了。



经济出问题，欧盟内部凝聚力也出现了严重问题。在过去，欧盟宣传自己，喜欢说“只有人想进来，没有人想离开”。2020年，英国退出欧盟，彻底打破了这个神话。此外，最近二十年，欧盟没有做出任何有深度的改革，欧盟的布鲁塞尔总部被看做一个陈旧的官僚机构。新兴起的民粹政党，都把欧盟当作靶子，煽动脱离欧盟。2025年的今天，欧盟想单独和美国打贸易战，在理论上都没有可能性。



欧洲新崛起的民粹政党，在总结欧盟衰落原因的时候，喜欢把责任归结到移民身上。



<font color = "indigo">督工，欧盟的移民群体，真的起到了负面作用吗？</font>



移民问题，只是欧盟经济衰落的一个表现。



客观来说，最近十年欧盟接纳移民，确实造成了很大的财政压力。但是消化移民，曾经是欧洲国家的强项。上世纪五十到七十年代，欧洲国家很多殖民地独立，大量殖民者后代回归本国。这些人的生活习惯、身份认同，跟移民区别不大。德国这样没什么殖民地的国家，也从土耳其主动接纳劳工，解决劳动力短缺问题。甚至东德引入的14万越南移民，也完美融入欧洲。



现在的欧洲人，对移民群体观感不好，根本原因还是工作岗位不够。这些新移民一方面没有收入，消耗社会福利，一方面又没有工作，形成了治安隐患。失业率高的时候，再先进的社会，也解决不了人口融合问题。



欧洲失业率高，原因是缺乏新产业。2007年全世界都嘲笑的美国互联网产业泡沫，事实证明就是美国下一个经济突破口，也是欧盟错过的最大发展机遇。



一般提起日本的发展，都会说日本经济“失去的三十年”。进入21世纪以后，欧洲也有“失去的二十年”。本来移动互联网兴起之前，在传统的IT产业，欧洲只比美国落后一点。比如说3G通讯、工业产线，以及工程软件，欧洲都有世界一流的企业。诺基亚、爱立信，曾经都是世界通讯巨头。西门子的工业设计软件，现在还占据行业主导地位。2001年我买第一部手机，是法国的阿尔卡特。



但是进入21世纪，欧洲错过了互联网的每个风口。作为“世界前二”的超大型经济体，欧洲不仅没有微软、英伟达、谷歌这样的全球企业，也没有像腾讯、阿里、美团这种内部大企业，甚至不能给欧盟内部市场提供互联网服务。



缺乏本地互联网平台，所以欧洲互联网普及程度越高，就有越多财富流入美国和中国。最直观的表现是广告。以前的企业投放广告，会选择各个城市的地标建筑、巴士站台、灯牌，以及电视台。现在娱乐集中在互联网，企业选择在推特和Youtube和Tiktok花钱，这些钱最后还是流向了美国和中国。



新风口分不到钱，欧盟的优势领域，也在逐渐被压缩。现代工业，包括手工业在内，都必须应对互联网转型，才能保持现有地位。移动互联网、人工智能和新能源，这几项新产业是彼此相关的，但都是欧洲的弱项。所以欧洲的传统优势产业，比如说汽车、芯片、能源，市场份额也开始下降。



<font color = "indigo">欧盟不缺市场、资本和人才，冷战结束以后还节省了上万亿的军费，为什么几十年都没孵化出自己的高新技术企业？</font>



因为扩张以后的欧盟，几乎失去了政策执行力。



欧盟扩张前，只有十多个成员国。即使欧盟内部有“不合群”国家，比如说英国就经常公开唱反调。但是依靠德法两国的体量优势，还是可以推动经济政策落实。

前面提到过，九十年代开始，欧盟快速扩张。一开始，扩张给欧盟带来了很多好处。当时日本遇到了劳动力紧缺的问题，不得不推出技能实习生制度，跑到东南亚找劳动力。欧盟的新加盟国家，有将近一亿总人口，给欧盟提供了几千万优质劳动力，轻松解决了产业规模问题。



加入欧盟的东欧国家虽然不如西欧，但是从全球来看，也属于一只脚迈进了发达社会的国家。这就为西欧资本扩张，提供了目的地。从九十年代开始，东欧大部分产业，都被西欧资本收购，把东欧国家变成了西欧资本的后花园。对东欧来说，有西欧国家支持，经济转型起码比前苏联国家顺利，所以双方都比较满意。

但是等到经济增长变缓，欧盟扩张的后遗症出现了。以前欧盟国家内部意见不同，德国、法国还可以用体量施压。现在欧盟有20多个成员国，谁说话都不管用，任何政策都要经过漫长博弈。



就拿互联网产业来说。欧洲很多政治精英都发现互联网平台的重要性了，希望能建立贸易保护机制，培养欧洲企业。比如说法国、意大利和西班牙，从2017年开始，就要求欧盟征收“数字服务税”。如果法案落地，谷歌、推特这些互联网巨头，在欧洲产生的所有收入，起码要缴税3%。这笔钱，可以用来扶持欧洲自己的互联网平台。



这个政策看起来对欧盟有利，结果很多年都没能形成决议。最大的反对者是爱尔兰，担心这个政策影响自己的避税港地位。波兰这些东欧国家，害怕得罪美国，也投了反对票。所以从2019年开始，法国抛开欧盟，自己独立征收数字税。

到今年，欧盟不仅没能推出数字税，跟美国谈判贸易协定的时候，还宣布要取消欧美数字贸易壁垒。换句话说，欧盟已经彻底放弃和美国互联网产业竞争了。 

官方态度这么软弱，互联网人才和资本，就不可能选择欧洲，也不可能振兴欧洲。新的欧美贸易格局成型以后，欧洲的经济只会继续下滑，更多的民粹政党，会把欧盟当作攻击对象。



欧盟的前身，是欧洲煤钢共同体，最早的发起者，就是法国和联邦德国。去年底，德国右翼政党选择党，开始鼓动德国退出欧盟。9月10日，巴黎也爆发了示威活动，参与者有十多万人。这些抗议者其中一个诉求，就是法国退出欧盟。



欧盟创始国和主要领导者，都出现了脱欧思潮。比起发掘新产业，欧盟现在更应该做的，解决德国和法国为什么要维持欧盟的问题。



把视野放宽到全世界，美国对欧盟签订一边倒的贸易条约，并不是特例。韩国、日本，和美国签订了类似的条约，其他发展中国家也默认了美国的单方面加税政策。特朗普正确地评估了美国当前的实力，就是可以随意碾压中国之外的所有国家。

过去十几年，我们生活在一个快速成长的工业国，相对美国经济的差距不断缩小，所以并不觉得美国经济的活力有什么了不起。但是过去20多年，美国3亿人口，至少领导了五次颠覆性的全球产业革命：互联网、页岩油，生物制药、人工智能软硬件、可回收廉价航天，平均每四到五年一次。如果不考虑中国，连电动汽车产业都可以算作美国的绝对优势。虽然美国社会内部还有很多问题，但是以20世纪的标准来看，美国没有停滞或者衰落，而是在加速度发展。



凭借连续不断的产业革命，美国对全世界的优势在扩大，再配合美国之前几十年积累的意识形态优势，美国对全球秩序的控制力也在增强。只是面对中国这个超级工业国的跟进跟超，美国优势才没有那么明显。中国在考虑国际问题的时候，一方面要正确评估自己的力量，另一方面，也不能因为和美国差距缩小，就忽视美国霸权的新型产业基础。
