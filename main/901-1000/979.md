#

大家好，2025年11月16日星期日，欢迎收看979期睡前消息。请静静介绍话题。



<font color = "indigo">10月15日，湖南省多家官方媒体报道，长沙一家名叫铜官竖店的微短剧影视基地开业。按照官方说法，长沙竖店建设了300多个室内拍摄场景，200多个室外拍摄场景，可以把微短剧拍摄成本压缩30%。

浙江横店是中国影视拍摄中心，现在湖南的短剧影视基地直接取名“竖店”，能复制成功吗？</font>



所谓“竖店”，就是拍“竖起来看”的手机短剧。所以要碰瓷大屏幕时代出道的横店。



如果这是一个新投资项目，不太可能成功。但好在“长沙竖店”本来是一个失败的古镇项目——铜官窑古镇，定位和张家界大庸古城、常德桃花源古镇差不多。按原来的轨迹发展，铜官窑古镇会慢慢变成废墟，最后政府收回土地，花钱拆除。现在古镇项目转型影视城，运气好可以有现金流，最差结果也就是继续当废墟，算是死马当活马医了。



长沙竖店的主要资产，是大片的仿古中式建筑，还有一些近代西方风格塔楼。这些建筑都是当年“铜官窑古镇”项目的一部分，已经有将近十年历史。所谓“铜官窑”并没有连续不断的历史传承，只是利用了断断续续的考古成果。

![图片](https://mmbiz.qpic.cn/mmbiz_jpg/Mo3q9dvIibMCMMbp1PdSCjmCUWHZEYFR3a6ZVMNbgd52RibzjDUHooLG3IjcibfBKsdEBibpq51d6OqHfN5d8Q8QQg/640?wx_fmt=jpeg&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=1)



上世纪50年代，考古工作者在长沙郊区铜官镇发现了一个唐代陶瓷遗址，取名叫铜官窑，名气不大。1998年，德国打捞公司在印尼发现了一艘唐代沉船，里边有五万六千多件瓷器，推测都来自铜官窑，具有阿拉伯风格。这艘船被叫作“黑石号”，铜官窑也成了唐代中国跟西亚贸易的重要遗址。



睡前消息[第926期](/main/901-1000/926)介绍过，当年全国都流行建古城，到现在有1000多个古城古镇。在铜官窑项目上，长沙市政府最聪明的选择，是不贪图“肥水不流外人田”，没有让国资直接包办，而是把土地拿出来，请外面的企业去做。



2012年，长沙市望城区政府找到了一家合作方，新华联集团。双方约定共同开发长沙铜官窑国际文化旅游度假区。2015年，铜官窑古镇正式开工。同一年新华联在股市筹资，表示要在铜官窑投资50亿，建8个博物馆，5个演艺中心、18个人文景点。按照新华联自己的说法，铜官窑古镇收入可以达到207亿，利润也有将近70亿。



如果只卖古城门票，显然赚不到这么多钱。新华联看好铜官窑古镇，真正的原因还是房地产。十年前上网的老观众可能还记得，中国曾经流行企业家俱乐部，其中资历最老的叫“泰山会”，1993年就成立了。这个“泰山会”的成员包括联想柳传志，复星郭广昌，还有巨人集团老板史玉柱。2005年，马云也加入了泰山会。

![图片](https://mmbiz.qpic.cn/mmbiz_jpg/Mo3q9dvIibMCMMbp1PdSCjmCUWHZEYFR3icEecmmd3pZjCZwzMjUQIr4ickbJM1MNvGDkLRI8ZKOvzjHvic2ygK4Dw/640?wx_fmt=jpeg&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=2)



新华联老板傅军，就是“泰山会”的创始成员，上世纪80年代在湖南省外贸系统当干部，在进出口贸易领域积累了很多关系。1990年，傅军辞职成立了华联贸易公司，往马来西亚卖农产品，又从马来西亚进口橡胶和木材。赚到第一笔钱以后，傅军觉得搞外贸太辛苦，想要搞快钱，看中了房地产。



90年代有个口号，“十万大军下海南，各大财团抢地盘”。当时90年代海南刚刚建省，大量资金涌入海南楼市，在人均工资二三百块的时候，把海南岛的房价炒到了7500块一平。《我爱我家》里边贾志新决定去海南闯荡，就是炒房去了。



海南的房地产泡沫，当时也波及到了琼州海峡北面的广西。傅军的钱放在海南抢不过其他官商财团，但是可以跟风到广西北海市炒房。很多当事人都回忆了当年北海房地产泡沫，请静静帮我读一段：



> <font color = "indigo">*1992年，全市注册登记的房地产公司800家，到1993年5月达到1160家，共有数百亿资金涌入北海，出现一场房地产开发大战，全市130多个投资百万以上的项目剪彩。*
>
> *进入设计阶段的大楼，20层以上的有130多栋，30层以上的有30多栋，最高楼56层的‘川西北大厦’，已经启动了奠基仪式。除了高层大厦以外，城区的写字楼、商住楼、公寓楼普遍开花。曾经有6个财团送上联名报告，要求建98层的大厦，要创全国之最。*
>
> *在北海的房地产最火热时，有个别人甚至由炒房地产变成炒红线图、蓝线图，一单几百、上千万的房地产交易根本不用去现地察看，双方交易只在图上完成。*</font>



1994年，国务院出台《关于加强宏观调控的意见》，收紧银行贷款，吹破了海南、广西的房地产泡沫。傅军在文件出台之前，高位卖出了北海地产，赚到一大笔钱。泰山会不少元老，比如说万通集团冯仑，以及刚才提到的史玉柱，当年也在周边炒房。傅军通过一起炒房的经历，打入了北京商人交际圈，参与创立了泰山会。



从此以后直到2002年，中国房地产市场整体比较低迷，傅军开始搞多元化经营，比如说引进三菱帕杰罗技术，生产猎豹越野车，又比如跟五粮液合作，生产贴牌酒“金六福”。这段时间，新华联旗下既有房地产，也有矿业、化工和陶瓷，涉及10多个行业，让傅军挣了不少钱。



但是真正让新华联做大的，还是老本行房地产。从2002年到2018年，第二波房地产高峰期，傅军带着第一次的经验和资本，全面投入，一度70%营收都来自房地产。通过卖房赚的钱，傅军又投资了100多家公司，包括ofo共享单车、乐视汽车，都是泰山会老兄弟的项目。



2012到2014三年时间，中国房地产价格出现了一段平稳期。新华联大量地产集中在三四线城市，有点卖不动了，傅军开始思考转型，看中了文化旅游。在当时很多地产商看来，文旅项目可以从政府借钱，还可以低价拿配套住宅用地，可以说是稳赚不赔。铜官窑古镇是傅军投资的第一个文旅项目。几乎同一时间，万达也从住宅和商业综合体转向文旅，在全国建了13个旅游区，恒大建了15个。



一开始，官窑古镇表现得还可以。2017年古镇还没有开业，全国就有400多个商人赶到长沙，竞拍商业门面。不到两个小时，所有商铺全卖出去了，成交额5个亿。第二年旅游区正式开业，每个月要接待10万名游客。



新华联也把铜官窑古镇当成招牌，到处推销自己的成功经验。2018年9月，睡前消息另一位老朋友，四川阆中市政府跟新华联签约了合作协议。按照计划，双方要一起投资100亿元，共同开发阆中古城。新华联对外表示，要把长沙铜官窑古镇成功经验，复制到阆中市。



至于说阆中古城项目做得怎么样，首先欢迎看我们2022年的[第455期](/main/401-500/455)节目。到了2023年互联网突然出现一篇文章，标题是[《看四川阆中如何搞烂自己的营商环境》](《看四川阆中如何搞烂自己的营商环境》)。我给大家念一段：



> *2019年8月，政府股东经市财政担保向合资公司借款，至今本息合计1800余万元，且逾期三年拒不归还，致使企业现金流短缺陷入困境。*
>
> *阆中政府自身财政困难，为了拿到古城景区门票的经营权融资，于2022年1月15日，单方面收回古城景区特许经营权。*
>
> *按约定，合资公司应由新华联现金出资占股70%，阆中市政府以土地注入占比30%，但政府应该注入的土地资产，一直没人办理。*



这篇文章显然是站在新华联立场上的单方面说法。我不关心阆中市和新华联谁对说错，但可以确认，阆中古城项目肯定失败了。去年新华联还在起诉阆中市政府，索赔违约金4560万。



铜官窑古镇也没比阆中古城好到哪儿去。现在打开一些订票软件，还可以看到古镇门票价格，原价200，打折以后也要168。点开评论区，基本都是游客的抱怨，认为自己花了冤枉钱看人造建筑。开业第二年，铜官窑古镇又遇上了疫情，跟全国绝大多数旅游区一样，坐等亏钱。



旅游项目就算亏钱，不代表就要卖掉。[928期](/main/901-1000/946)介绍廊坊新奥集团投资《只有红楼梦》实景演出，回本周期在三十年以上，每年收入往往覆盖运营成本。但是背后新奥集团拥有天然气特许经营权，每年可以保持千亿营收，还能亏得起。铜官窑古镇的问题，是新华联撑不住了。



前面提到过，新华联当年建设铜官窑古镇的钱，是从股市借来的。之后五年，新华联又在全国投资了四个旅游项目，一共借了200多亿。到后来，某些债券每年利息10%。2019年有192亿债务集中到期，新华联把全部现金拿出去应付债务。



刚刚还完钱，疫情开始了，全国房地产价格进入了下行周期。等到2020年，新华联有10亿债务到期，这个时候公司已经拿不出钱，构成债务违约。新华联的房子都集中在三四线城市，实际价值可能也就是两三千块一平米。新华联不愿意按这个价格清算，违约债务越来越多。到了2022年，新华联违约金额达到了26亿，资产负债率将近96%。大量债权人找到法院，申请破产重整。



新华联老板傅军毕竟见过20世纪的第一波海南房地产泡沫，凭借丰富的经验，他开始割肉止血，先放弃还有土地价值的省会附近古镇。2023年11月8日，铜官窑古镇的运营公司新华联文旅向北京法院申请破产重整。随后一年，长沙市政府牵头债务重组，找了5家投资人注资20亿，接手古镇项目。



从此以后，铜官窑古镇的运营方，换成了一家叫盈新资管的公司，具体的运营公司从新华联文旅改成盈新发展。接手古镇重组第一年，盈新发展就宣布扭亏为盈。按照公司的业绩预告，2022年亏损将近35亿，2023年可以赚3到4亿。但是仔细看财报，所谓盈利只是会计操作。所谓盈利，是盈新发展重整以后，有将近40亿债务不用还，按照会计准则算成“收益”。今年8月，运营公司发布最新财报，铜官窑古镇本身还是个现金黑洞项目，半年亏损4个亿。



新公司接手铜官窑古镇，也不是有多看好文旅项目。盈新的老板叫王赓宇，之前从来没有管理过任何实体产业，从人民银行总行辞职以后，一直在搞资本运作。王赓宇收购新华联文旅，净资产只有1000万，属于典型的蛇吞象并购。对王赓宇来说，拿下新华联文旅不是为了亏钱的古镇，而是上市公司的壳。有了上市公司，就可以借着全面牛市的机会，用概念换钱。



就在上个月，10月21日，盈新发展发布公告，表示自己要收购一家广东半导体公司，以后生产内存条。早在公告发出之前，这家公司股价就涨停了两天，等到正式公告发出来，又涨停了三天。



对盈新发展名下其它资产，王赓宇也不太上心。前面提到，铜官窑今年刚刚亏掉了4个亿。这家公司在西宁、株洲、海南的旅游业务，也亏掉了将近4个亿。同时，盈新发展在中小城市还闲置了将近75万平米的房子，不知道该怎么办。



虽然股市圈钱容易，但也经不起每年8个亿的现金外流。所以铜官窑古镇彻底放弃了文旅，转型“竖店”短剧影视城。也许短剧赛道并不好走，但只要能把古镇经营成本放下，或者稍微赚一点房租钱，就比现在强。



<font color = "indigo">竖屏短剧最近几年很火，湖南竖店能抢走浙江横店的热度吗？</font>



2000年横店出名以后，中国各地都开始建影视城，基本上拍一部古装影视剧，就要建一座基地。到2013年，中国已经有1000多座影视城，跟现在的古城数量差不多。北京大学文化产业研究院调查发现，这些影视城80%都在亏损，15%勉强收支平衡，只有不到5%能赚钱。当时有个影视城老板抱怨恶性竞争，我给大家念一下：



> *“从春秋到汉朝，一直到民国，哪个时代的古建筑都有了。目前国内的影视基地已经严重超载，80%都是多余的。雷同也造成了场租费的恶性竞争，你一万，我八千，你八千，我五千。未来20年不再新增影视基地，已经足够影视剧拍摄需要了。”*



 更重要的是，湖南“竖店”要做短剧，最大的对手不是横店，而是其他“竖店”的江湖地位。现在搜索“竖店”，搜索引擎默认是郑州郊区的影视城，其次是上海郊区的拍摄基地，湖南这个竖店根本没有热度。



郑州能在“竖店”短剧产业做到第一名，优势不是仿古建筑随便用，也不是政府引导支持，而是在人工便宜的地方，让市场自发筛选出合适的产业聚居区。



对短剧观众来说，场景漂不漂亮、建筑是否还原都不重要，甚至优秀的剧本也不是关键性因素。因为竞争太激烈，成功的随机性太强。刚刚做出爆款产品的团队，让他们按照上一次的思路，再来一遍，也很难保证成功。

![图片](https://mmbiz.qpic.cn/mmbiz_jpg/Mo3q9dvIibMCMMbp1PdSCjmCUWHZEYFR3s6bum44crA5WKq3214gOfYqwErn3GXprtffUyZ2zcfkMcTdwUeYNcA/640?wx_fmt=jpeg&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=3)



所以短剧的打法是高频试错，快速产出大量不同类型的产品，没人看就下架，十几次试错有一部赚钱就能覆盖其它作品成本。这种盈利逻辑，导致短剧公司对成本非常敏感，只要可以节省开支，在哪里都能拍，演职员基本都是临时工。



河南是中国农村劳动力供给最过剩的地方，十几年前就建成了国内最大富士康厂区。在短剧兴起之前，郑州凭借人口多、工资低，形成国内小广告拍摄基地。这些人既会剪辑，也可以写剧本，跟短剧生态高度重合。



除此之外，郑州还自带影视配套产业链。上世纪90年代，河南许昌市鄢陵县张北村的村民进入影视圈做灯光师，后来带动全村人入行。这些人积累了资本和关系以后，经营灯光设备租赁，比剧组自备更便宜，更可靠。现在全国影视行业80%灯光师，都来自张北村，影视城里边的灯光设备店，老板也基本都是张北人。



在短剧时代，甚至郑州的烂尾楼也是一个优势。请静静帮我读一段新闻：



> <font color = "indigo">*一个短剧拍摄现场，所有信息都被记录在一则15秒的视频里。有人把这则视频转发在了郑州一个名为“养生谷”的小区业主交流群中，一名业主疑惑道：“我怎么看像在售楼部门口。”*
>
> *去年4月，任志万第一次在业主交流群里知道，自己的小区售楼部变成了“拍短剧的地方”。他购买的房子原本应于2022年交付，至今已逾期1000多天，房企爆雷、房子烂尾也从受人关注的事件，逐渐成为媒体上的旧闻。他没有想到，荒废多年的烂尾楼，竟然以短剧的方式“重生”了。*
>
> *郑州短剧从业者李霄（化名）说，租赁拍摄场地是短剧制作必须支出，通常占到核心成本的10%-20%。如果场地需求稳定，出租售楼处对楼盘来说是一笔“不错的流水”，尤其在房地产市场调整期的当下，“售楼处和别墅的场地费一般在几千元一天，一个月几万元，够给几名楼盘员工发工资了。”*</font>



郑州现在有15个短剧拍摄基地，基本都是闲置楼盘改建。再加上农村提供的大量低薪工人和廉价楼盘，这才是交通枢纽郑州产生“竖店”产业的原因。行业内估算，郑州微短剧拍摄成本相比其他城市，要低10%-20%，如果算上后期制作成本能低40%。拍一部100集的微短剧，在郑州只要投入30万，每一集的成本，甚至低于一辆电动车。



2023年短剧刚刚火起来的时候，郑州已经可以年产1100多部，占到全国的1/3。去年10月，郑州市政府才发现自己的辖区内有短剧基地，出台扶持政策。反过来说，如果让政府主动去设计一个短剧基地，恐怕做不出来主动利用烂尾楼的事情，成本优势也就没了。



湖南竖店的对手，不止是郑州。从2024年开始，西安就在跟郑州竞争“短剧之都”的地位，高峰时期产出短剧占到全国60%。深圳文旅局7月也推出政策，表示可以直接发钱给制作商，补贴30%成本。湖南拿一个亏损古镇，就想形成短剧产业链，显然没什么竞争力。



最后还有一个更加重要的问题，短剧的产业规模可能不会再扩大了。这次Sora版本更新以后，相信很多观众都刷到过猫做饭或者狗开车的视频。我还刷到一条评论，说自己奶奶刷抖音，看见猫做饭以后问自家猫为什么不行。这说明现在的AI视频，已经接近了廉价短剧的画面水平，试错成本显然低于真人现场拍摄。



所以在AI视频彻底成熟以前，第一个受到冲击的领域，可能就是短剧市场。为了快速试错，大多数次要场景都可以交给AI。过去十几个人的团队，可能保留五六个人就够用了。未来几年，短剧产业可能会扩大，短剧拍摄基地的需求反而可能下降。



湖南现在才想到建“竖店”，唯一的优势，或许就是之前的古镇亏得太厉害，预期比较低，只要有一点现金流就好。但是，全国废弃的古镇太多，有保安认真巡查的很少。我的亲身经历证明，三五个人的小剧组，如果只是想低调进去借个场景，根本就不用特意去什么影视城。所以，长沙的“竖店”，首先还是要解决剧组进驻问题。
