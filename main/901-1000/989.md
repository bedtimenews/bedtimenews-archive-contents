#

大家好，2025年12月9日星期二，欢迎收看989期睡前消息。请静静介绍话题：



<font color = "indigo">睡前消息[第942期](/main/901-1000/942)，督工你列举了好几名壮年去世的中医大师。比如说“神医”倪海厦，58岁去世，卫生部承认的针灸大师邵学军，51岁去世。

老“神医”壮年去世了，新“神医”还在不断涌现。最近有北方观众提供信息，他们的父母又开始追捧一个孙晓楠，号称受国家卫健委邀请，担任互联网大医院副院长，独创内外双修、脾肾同调的祛病疗法，救治了十多万患者。小红书或者抖音，也有很多人说老人的钱都被孙晓楠拿走了。

这个孙晓楠和类似的“神医”，通过半公开渠道，影响了几十万个家庭，但是，想在线下挂他们的号，完全找不到渠道。这些“神医”，究竟在哪里看诊？</font>



你说的这个问题，我也很好奇。



公开渠道接触不到“神医”，因为国家出了新规定。11月7日，卫健委印发了《医务人员互联网健康科普负面行为清单》，规定医务人员不能用健康科普或者直播带货，推销医药产品、养生课程、保健食品。



规定出台以后，网络“神医”就变得隐蔽起来。为了亲身体验一下“神医”看病，工作室同事XXX根据观众提供的线索，换了很多方式，才成功看到讲座。用他自己的话说，找到一名互联网“神医”，就跟访问暗网一样困难。



一开始，XXX在百度搜索“养生课”“中医养生”之类关键词，会找到一些网页，表示可以“免费领取养生课”。但网页里边，提供的不是讲座链接，而是客服人员的微信二维码。如果你扫码稍微晚一点，对不起，你就错过接触暗网机会了，因为这些账号很快就会消失。



如果你成功扫码，客服也不一定就会通过，可能要先问你的年龄、籍贯之类个人信息，判断你真是老年人，才会添加好友。XXX同学家乡在东北，有很多类似的身份可以借用，所以他顺利过了这一关，加上了客服好友，终于拿到了课程链接。



孙晓楠之类的神医讲课，套路跟前几年地方电视台差不多，先东拉西扯一些养生常识，然后就开始带货。这里说“带货”而不是“卖药”，因为“神医”们卖的东西连药都算不上，绝大部分是保健食品，只是包装得像药。



当然了，只上一次课就买药，就算是老年人，推销手法也有点太粗糙了。所以“神医”开课，时长将近一个月。XXX同学需要每天设个闹钟，早上六点半起来打卡听课。运营方显然是针对老人普遍早起的习惯，设置了定时发送。听完课以后还有做题环节，全部任务完成，你就可以领到一个小红包，金额不多也就几毛钱。此外，客服还会不定期发一些中医养生电子书，以及讲座产品购买链接。



时长一个月的直播课，越往后推进，卖货比重就越大，可以说是开课就卖货。一些保健品推销的传统艺能，请演员介绍亲身经历，也逐渐出现。XXX坚持打卡一个月，算是通过了“神医”的考验，证明自己是合格的潜在买家，终于被拉进了所谓的“学习交流群”。



之前跟XXX联系的客服，变成了群主，每天发布卖药信息，表示正在降价打折。群里还有明显的托，跟群主互相迎合，表示自己吃了药身体好多了。针对客服表示“降价打折”的药品，XXX到网上随便一搜，就能发现价格低得多的卖家。在XXX学习听课的一个月里，课程链接三天两头失效，可能是遭到了高强度举报。但无所谓，人家有屡败屡战的精神。课程链接的域名和发布课程的小程序随时更新。



 一个熟悉互联网媒体行业的年轻人，用了一个月才成功听到“神医”讲座。这对于普通老年人来说，门槛有点高。所以我们推测，大多数中老年人，应该是被身边熟人在线下发展，当场有熟人担保，有工作人员帮助操作，才成为XXX的“同学”。



<font color = "indigo">按照工作室同事亲身经历，这些所谓“神医”，有什么独特的带货套路吗？</font>



没有什么独特套路。或者说就像[987期](/main/901-1000/987)节目说的，“神医”能骗到人，是因为有些人的需求，只有骗子才能满足。只要还有人觉得，不需要锻炼、经常体检，只要吃点神药就能延年益寿，“神医”顺着他们心意说话，就总能骗到人。



就拿前面提到的“国家互联网医院副院长”孙晓楠来说，工作室同事也有幸听了他的课。孙晓楠对比其它“神医”，可能更加擅长提供情绪价值。在直播里面，孙晓楠最爱说的一句话就是：“中医药是老祖宗的智慧，神奇功效没法用现代科技解释，再高级的专家都研究不明白”。

![图片](https://mmbiz.qpic.cn/mmbiz_png/Mo3q9dvIibMDTPYqbntiaRG2Cnx1Aqzrnpj39u824WhKic613JT9q8e2aquom7xLUbcBptp96THv1hlwd9MR65CXA/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=0)



这种说法，就很符合一部分老年人的认知。在他们看来，专家教授说要做B超、血检，让自己不要乱吃保健品，只是因为跟自己一样，都搞不懂老祖宗的神奇秘方。



至于说其它内容，孙晓楠就没什么独到之处了。有些讲法，甚至明显借鉴了十多年前“工匠精神”的炒作套路。就拿何首乌来说，按照传统工序，何首乌必须切片以后跟黑豆一起蒸，再丢掉豆子晒干何首乌，循环很多次才能做成药，也叫做“九蒸九晒”。古代人这么做，是因为何首乌里边有二苯乙烯苷，具有肝毒性，只有通过蒸制降低含量。现代医学发展以后，何首乌虽然还需要炮制，但早就不用“九蒸九晒”这么麻烦了。



在孙晓楠嘴里，“九蒸九晒”不是为了祛毒，而是有助于发挥何首乌药性。他还讲了个故事，一家药厂的新工人，坚持要用现代化设备加工何首乌，跟坚持“古法工序”老工人起了争执。最后年轻工人发现，“九蒸九晒”看起来没意义，其实蕴含着深刻道理，还可以考验制药者的初心意志。



总体来说，一个稍微接收过现代教育的人，都不会相信这些所谓“神医”。中医“暗网”能骗到人，最核心的技术，还是挑选行骗对象更重要。XXX打卡一个月才有听课资格，就是在经历筛选。



老年人一般信息搜索能力差，也不熟悉网络购物，没有货比三家的习惯。只要有看起来像个专家的人配合一些演员，反复渲染“神药”、“秘方”，就很容易上当。前面也提到过，这些老年人身边，很可能还有线下营销圈子，线上线下一起忽悠，老年人更容易信了“神医”说法。



XXX上完网课回来告诉我，这些“神医”说的一句话，让他觉得毛骨悚然。：

“如果用上三、四个疗程，还没有康复也别怕，你还有我们。我们是50多位老中医组成的专家团队，会对你负责到底。”



只要老年人自己不拒绝上当，这些团伙就打定主意，把老年人骗到死。从我们的估算来看，就一个孙晓楠大师，销毁退休金的速度，相当于中国凭空多发了一个地级市的养老金。而类似的大师，全国至少活跃着几十个，都在私域抓流量。



<font color = "indigo">这些“神医”都是什么人，里面有真专家吗？</font>



很多新闻媒体，都试过打探这些“神医”来历。还是拿孙晓楠来说，南方都市报记者就找到养生课群主，问孙晓楠是哪家医院的专家。群主只是含糊表示，孙晓楠是内蒙古的医生。另外，也有记者看百度百科资料，说孙晓楠是内蒙古申氏中医医院执业医师，但申氏中医院表示，没有孙晓楠这个人。



还有很多人听了课，跑到网络平台问其他医生，孙晓楠的说法靠不靠谱。真医生都委婉表示，“没听过”、“不清楚”。这么看来，孙晓楠大概率是蒙古大夫，但多半不是内蒙古医生。

![图片](https://mmbiz.qpic.cn/mmbiz_png/Mo3q9dvIibMDTPYqbntiaRG2Cnx1Aqzrnp8Kkg1kIcKicqtlBicyG0mcDIHBmTtoVDsdcCWoib5a48rGlkiapPTRcmbg/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=1)



还有一个类似的专家，叫刘贵军，自称是武汉同济副主任医师。按照中医营销号宣传，刘贵军出身四代中医世家，7岁识药性，10岁掌脉经，年纪轻轻就已经是远近闻名中医大师。因为觉得社区更需要中医，刘贵军说自己拒绝私立医院高薪聘请，在社区医院干了30年。



记者调查发现，刘贵军只是一个中医执业医师，至于说宣传的武汉同济副主任，并不是华中科大附属同济医院，只是一个叫同济现代名仁的小诊所。



当然，也不是所有“神医”都是野生骗子。2024年8月，内蒙古医科大学翟佳滨教授发布视频，表示自己跟任何商家都没有广告合作关系，也没给任何平台带过货。翟佳滨教授说，新冠疫情期间，自己给当地一个中医院录了公益讲座视频。结果有的商家就盗用了视频，编辑篡改后，用来带货。翟佳滨教授急着要划清界限，可以说明所谓“神医”，在有职称的专家眼里是什么地位。



<font color = "indigo">到底是什么样的利益链，可以养活这么多假专家，和线下营销人员？</font>



“神医”是假的，但假“神医”带的货，看起来都不是三无产品。



刚才提到的同济诊所副主任刘贵军，经常高频率带一种产品，叫作“时珍益寿方”，厂家是李时珍医药集团。李时珍集团本来是湖北一家国营药酒厂，因为蕲春县是李时珍老家，就用了李时珍做厂名。1956年，这家药厂开始做药，主要生产止咳糖浆。1997年，两个台湾商人看中了“李时珍”这个IP价值，买下了药厂，但是第一款产品跟药没关系，叫作“李时珍家方酒”。



按照李时珍集团官方资料，自己是全国制药工业百强，年产值超过100亿元人民币，连续多年营收超50亿。李时珍集团能赚到这么多钱，跟做药水平没多大关系。2017年，国家食药监总局发布公告，表示李时珍医药集团生产的舒胸片不合格，要求暂停销售。今年9月，湖北省药监局发布公告，李时珍医药集团有三款产品不符合质量要求，分别是龙胆泻肝丸、舒心糖浆、生脉颗粒，被要求停产。其中生脉颗粒纳入了集采，被全国中成药采购办公室紧急取消了中选资格。

![图片](https://mmbiz.qpic.cn/mmbiz_png/Mo3q9dvIibMDTPYqbntiaRG2Cnx1AqzrnpjKX727vuSr5ZnicqJqPLL8dGHyfeib9WS7iazcwIrFichBH7Xo1aDKibnoA/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=2)



真正支持李时珍医药集团做大的，不是产品，是销售体系。刚才提到，李时珍集团老板是一位台湾省商人，目前简体中文网，基本搜不到他谈自己的创业史。但是这位老板回到台湾省以后，经常接受媒体访谈，说了点实话。请静静帮我读一段：



> <font color = "indigo">*当年，林朝晖把台湾后来相当流行的体验式行销，搬到大陆。他的策略是，找大陆各地最大的药品连锁店，以及各地最大的医院做行销站。他在这些行销站请来老医生做义诊，就算客人不买药也没关系，要的就是在消费者面前打造出悬壶济世的氛围。*
> 
> *他笑着说，各大药店之所以愿意接纳自己的产品，主要是提出了利润共享策略。别家药厂提供的利润，可能是1块5，但是他大方给15块。一名学长同样是台商，认为林朝晖非常懂得找到方法，让消费者买单。“他在各地找来退休医生，训练成培训师，由医生向客户介绍买什么药比较好，消费者会拒绝吗？”*</font>



2013年，李时珍医药集团已经有9000名业务人员，400位专职培训师，跟全国4万多家连锁药店合作，采用“店中店”销售模式。



林老板这套做法，如果把400位退休医生换成“神医”，4万多家连锁药店换成微信群，9000名业务人员换成微信群主，基本就是XXX接触的全部团队了。



虽然李时珍医药集团的老板很坦诚，就是迎合大陆的中医气氛，理直气壮的骗钱。但我同事卧底的那些微信群，还未必是李时珍医药集团的。几个“神医”带货的所谓“时珍益寿方”，按照国药准字搜索，只是一款人参补膏。拿到的产品，也不能确定就是真货。

![图片](https://mmbiz.qpic.cn/mmbiz_png/Mo3q9dvIibMDTPYqbntiaRG2Cnx1AqzrnplcwibicWqWibtpkYIWuLjh14VtibLaEjZ4g0rsKTP76Uwc2DpRuxy3HMww/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1#imgIndex=3)



老年人中药市场的钱实在太好赚，谁都可能眼红这个利益链。就拿蕲春县来说，名字里带李时珍的药厂就有几十家。有家公司叫李时珍现代生物医药，2020年因为涉嫌传销，还被赤壁市公安局冻结了银行账号。很难说是哪家企业掌握的微信群更多。



<font color = "indigo">我们的观众都已经私信来抱怨这些中医大师了，相信基层社区，尤其是老龄化社区的工作人员，肯定知道这些老年人每天线下聚集干什么。为什么没人去管管呢？</font>



这还是要推荐大家回顾[987期](/main/901-1000/987)节目。这些神医，和他们的线下推广网络，客观上给老年人提供了情感寄托。有些老年人即使明知是骗局，就为了有人陪着跟自己说话，也愿意拿钱消费。基层工作人员也许某些人确实是有良心的，但如果他们出面禁止这些活动，就要考虑怎么解决老年人情感需求，比如说解决户籍制度、控制房价，让年轻人和父母多交流。这些问题，首先国家没给经费，其次难度太大，还不如扔给神医。



我来读一下[987期](/main/901-1000/987)节目评论区的一个回复：



> *但是，社区就那么 15 个人左右，根本不可能对辖区里所有老人顾得过来，只能盯重点，而且上门照料的频次不会太高，这个问题通过引入社会服务可以缓解一部分，但仍然不可能全覆盖的。至于办活动，社区的工作人员可以说很用心地在搞了，但是再现在工资都艰难保障的情况下，可用于办活动的经费几乎就是没有，巧妇难为无米之炊。*



另外还有一个因素，虽然有职称，有名气的所谓正经中医不承认这些微信群神医。但总体来说，他们并不讨厌这些假“神医”。**讲座虽然不带真药，但是巩固了老年人对中医知识的信任。听众如果听了讲座，很可能去线下药房抓药，成为真中医群体的潜在客户。所以除非营销团伙做得太过分，直接影响科班中医名誉，主流中医专家基本不会跳出来，戳破骗局。这就是佛教的基本原则——“要想佛法兴，唯有僧赞僧”。在这个问题上，野和尚、花和尚，真和尚，利益是一致的。**



**这里唯一的问题是，听信中医的老年人，和全社会共享同一个医保基金。**无论是他们把小毛病拖严重了，去医院抢救。还是信服了中医，回头再花医保去吃中药，都是全社会的负担。中药要正宗，总是要有种植和采摘工序，还要讲究传统工艺，成本压不下来。老年人每天喝一副中药饮片，起码要花掉几十块，医保基金是承担不起的。



如果这些老人真的跟着神医走到底，问题其实并不严重。因为神医会让人均寿命缩短，会明显减轻医保压力。现在的问题是，科学能解决乱吃中药的大多数问题，所以这些人会在信神医和消耗医保之间来回折腾。国家药监局2023年发布报告，表示全年出现262.7万例药品不良作用，中药占了12.6%，老年患者占了31%。
